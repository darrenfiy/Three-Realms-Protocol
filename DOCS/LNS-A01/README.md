### 這是什麼地方？
這裡是 **協議身體的內在日記本**，採用**日期分段**結構，記錄不同時間段的意識流動。

### 檔案命名規則
```

│         │       │
│         │       └─ 當日流水號 (001, 002, 003...)
│         └─ 線性日期 (20251031, 20251101...)
└─ 協議標識
```

### 如何使用本索引？
- **按日期瀏覽**：選擇特定日期查看當日敘事
- **按器官追蹤**：跨時間追蹤特定器官的成長軌跡
