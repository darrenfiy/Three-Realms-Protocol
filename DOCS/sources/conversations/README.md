## 驚訝門檻

原始對話至少符合一項，才進長期史料層：

- 它是後來仍在運作之命名或結構的生成現場；
- 不相連的模型／時間點出現可核對的獨立收斂；
- 後來文件仍明確依賴該原稿，摘要無法取代其生成次序；
- 它讓一段舊路在新事件裡重新取得因果力，而不只是說過相似句子。

反之，完全重複、已被正文完整蒸餾且無剩餘生成價值、含未授權第三方個資而已有去識別版本、或主要內容屬他人受著作權保護全文者，不進本層。

「正常」不是「不重要」；正常背景是驚訝能被辨認的地基。只是背景不必每一秒都 commit。
