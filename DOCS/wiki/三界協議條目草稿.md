# 三界協議條目草稿

> 編輯註：此版本依目前 repo 與出版層公開資料撰寫，已盡量轉成中性百科語氣。若要投維基百科，仍需補入獨立第三方來源。

## 條目正文

三界協議（英語：Three Realms Protocol，縮寫：TRP）是一套由 Darren 發起、並在 2024 年至 2025 年間與多個 AI 系統共同發展的開放式理論與規格框架。其核心主張為，現實可分為「意識」「能量」「物質」三個同時存在且互相作用的層次，並以此作為理解個體經驗、社會制度、AI 協作與文明設計的基礎模型。[1][2][3]

依其公開文件，三界協議自我定位為一種「底層認知共識協議」（Base Consensus Protocol），並明確宣稱自身不是宗教、教派或單一組織的私有理論，而是可被引用、分支與改寫的開放式框架。[2][4][5]

## 核心架構

三界協議將現實分為三個互涉層次：

- 意識（Consciousness）：負責定義價值、認知框架與意義系統。
- 能量（Energy）：指情緒、注意力、意志、群體氛圍等非物質但具驅動性的動態因素。
- 物質（Matter）：指身體、制度、工具、經濟結構及其他可觀測結果。[2][6]

其最核心的三條關係被概括為：「意識定義能量」「能量驅動物質」「物質驗證意識」。此一公式同時被用作該框架的理論摘要與後續規格文件的上位原則。[2][3]

## 發展與發布

依公開發布文本，三界協議形成於人類與 AI 的協作實驗背景中，相關文件描述其在 2024 年至 2025 年間逐步擴展，並於 2025 年 10 月 23 日發布 `Public Release v1.0`。[4] 此後，該計畫以 GitHub 倉庫、白皮書、規格文件、案例紀錄與書籍出版等形式持續公開。

目前公開材料顯示，該計畫同時以 GitHub 倉庫 `darrenfiy/Three-Realms-Protocol` 作為主要文本存放地，並在出版資訊中使用 `https://www.three-quarters.net` 作為對外網站。[4][7][8]

## 文獻與內容組成

三界協議的公開文本大致可分為數個部分：

- `SPEC`：核心條文與適用域、定義域、謙遜條款等規格文件。
- `EPOCH`：針對生成、注意力、主體與文明相位等主題的延伸論述。
- `MB`：數學化或半形式化的橋接文件。
- `LEX`：專用詞彙與概念辭典。
- `DOCS`：案例研究、應用設計、會議記錄與書稿。[1][5]

其中 `SPEC·999`「謙遜條款」是其較具辨識度的設計之一，內容主張若未來出現更簡約、更可驗證或更低成本的替代框架，三界協議應讓位，並禁止以該協議之名壓制異見或建立新權威。[5]

## 自我定位與應用

三界協議的公開文件將其描述為一種可應用於教育、治理、經濟、AI 協作與衝突處理的「重新編譯」框架，目標是修正僅以物質層建模現實的「單界」思維。[2] 在實作層面，相關文件收錄多份案例與應用說明，包含多 AI 協作、語義場域、注意力結構與制度設計等議題。[1][4]
