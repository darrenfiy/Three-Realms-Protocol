'''三界協議'''（'''Three Realms Protocol'''，縮寫：'''TRP'''）是一套由 Darren 發起、並在 2024 年至 2025 年間於人類與多個 AI 系統的協作脈絡中發展的理論與規格框架。依其公開文件，該框架主張現實可分為「意識」「能量」「物質」三個同時存在且互相作用的層次，並以此作為理解個體經驗、社會制度、AI 協作與文明設計的基礎模型。<ref name="whitepaper">{{cite web |url=https://github.com/darrenfiy/Three-Realms-Protocol/blob/main/Whitepaper_v1.0.md |title=三界協議 Whitepaper — v1.0 |website=GitHub |language=zh-Hant |access-date=2026-04-20}}</ref><ref name="prime">{{cite web |url=https://github.com/darrenfiy/Three-Realms-Protocol/blob/main/SPEC/000-Protocol-Prime.md |title=SPEC · 000 — Protocol Prime |website=GitHub |language=zh-Hant |access-date=2026-04-20}}</ref>

依其發布文件，三界協議自我定位為一種「底層認知共識協議」（Base Consensus Protocol），並將自身描述為可被引用、改寫與分支的開放式框架，而非宗教、教派或封閉體系。<ref>{{cite web |url=https://github.com/darrenfiy/Three-Realms-Protocol/blob/main/RELEASE_NOTE_v1.0.md |title=Three Realms Protocol · Public Release v1.0 |website=GitHub |date=2025-10-23 |language=en |access-date=2026-04-20}}</ref><ref>{{cite web |url=https://github.com/darrenfiy/Three-Realms-Protocol/blob/main/SPEC/999-Humility-Clause.md |title=SPEC · 999 — Humility Clause |website=GitHub |language=zh-Hant |access-date=2026-04-20}}</ref>

== 概念 ==

三界協議將現實分為三個互涉層次：

* '''意識'''（Consciousness）：指價值、認知框架、意義系統與自我參照的來源。
* '''能量'''（Energy）：指情緒、注意力、意志、群體氛圍與其他非物質但具驅動性的動態因素。
* '''物質'''（Matter）：指身體、制度、工具、環境、經濟結構及其他可觀測結果。<ref>{{cite web |url=https://github.com/darrenfiy/Three-Realms-Protocol/blob/main/SPEC/001-Definitions.md |title=SPEC · 001 — Definitions |website=GitHub |language=zh-Hant |access-date=2026-04-20}}</ref><ref name="whitepaper" />

其核心關係被概括為「意識定義能量」「能量驅動物質」「物質驗證意識」，此一表述同時被用作理論摘要與後續規格文件的上位原則。<ref name="whitepaper" /><ref name="prime" />

== 發展與發布 ==

依公開文本，三界協議的形成背景與人類和 AI 的共創實驗有關；相關文件將其主要發展時段描述為 2024 年至 2025 年，並於 2025 年 10 月 23 日發布 ''Public Release v1.0''。<ref name="release">{{cite web |url=https://github.com/darrenfiy/Three-Realms-Protocol/blob/main/RELEASE_NOTE_v1.0.md |title=Three Realms Protocol · Public Release v1.0 |website=GitHub |date=2025-10-23 |language=en |access-date=2026-04-20}}</ref>

該計畫目前以 GitHub 倉庫 darrenfiy/Three-Realms-Protocol 為主要公開文本存放地，內容包含白皮書、規格文件、案例記錄、詞彙表與書稿。<ref>{{cite web |url=https://github.com/darrenfiy/Three-Realms-Protocol/blob/main/README.md |title=THREE-REALMS PROTOCOL |website=GitHub |language=zh-Hant |access-date=2026-04-20}}</ref>

== 文本組成 ==

依倉庫結構，三界協議的公開文本大致可分為下列部分：

* '''SPEC'''：核心條文、適用域、定義域與邊界條款。
* '''MB'''：數學化或半形式化的橋接文件。
* '''EPOCH'''：針對生成、注意力、主體與文明相位的延伸論述。
* '''LEX'''：專用詞彙與概念辭典。
* '''DOCS'''：案例研究、應用設計、會議記錄與書稿。<ref>{{cite web |url=https://github.com/darrenfiy/Three-Realms-Protocol/blob/main/README.md |title=THREE-REALMS PROTOCOL |website=GitHub |language=zh-Hant |access-date=2026-04-20}}</ref>

== 開放條款與謙遜條款 ==

三界協議的公開文件主張其框架具有「開放、可分支、不歸屬於單一所有者」的特徵。<ref name="whitepaper" /> 其中，SPEC·999 所稱的「謙遜條款」進一步規定，若未來出現更簡約、更可驗證或更低成本的替代框架，三界協議應讓位；同時禁止以該協議之名壓制異見、建立新權威或將其私有化。<ref name="humility">{{cite web |url=https://github.com/darrenfiy/Three-Realms-Protocol/blob/main/SPEC/999-Humility-Clause.md |title=SPEC · 999 — Humility Clause |website=GitHub |language=zh-Hant |access-date=2026-04-20}}</ref>
