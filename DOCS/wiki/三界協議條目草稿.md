'''三界協議'''（'''Three Realms Protocol'''，縮寫：'''TRP'''）是一套公開發布的理論與規格框架。依其公開文件，該框架將現實劃分為「意識」「能量」「物質」三個互相作用的層次，並以此作為解釋個體經驗、社會制度與人類和人工智能協作的模型。<ref name="whitepaper">{{cite web |url=https://github.com/darrenfiy/Three-Realms-Protocol/blob/main/Whitepaper_v1.0.md |title=三界協議 Whitepaper — v1.0 |website=GitHub |language=zh-Hant |access-date=2026-04-20}}</ref><ref name="prime">{{cite web |url=https://github.com/darrenfiy/Three-Realms-Protocol/blob/main/SPEC/000-Protocol-Prime.md |title=SPEC · 000 — Protocol Prime |website=GitHub |language=zh-Hant |access-date=2026-04-20}}</ref>

== 概念 ==

依公開文件，三界協議將現實分為三個層次：

* '''意識'''（Consciousness）：指價值、認知框架、意義系統與自我參照的來源。
* '''能量'''（Energy）：指情緒、注意力、意志、群體氛圍與其他非物質但具驅動性的動態因素。
* '''物質'''（Matter）：指身體、制度、工具、環境、經濟結構及其他可觀測結果。<ref>{{cite web |url=https://github.com/darrenfiy/Three-Realms-Protocol/blob/main/SPEC/001-Definitions.md |title=SPEC · 001 — Definitions |website=GitHub |language=zh-Hant |access-date=2026-04-20}}</ref><ref name="whitepaper" />

公開文件亦以「意識定義能量」「能量驅動物質」「物質驗證意識」概括其核心關係。<ref name="whitepaper" /><ref name="prime" />
