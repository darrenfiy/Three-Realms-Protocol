## 編輯決議（已定案）

1. **佛佐全書退場。** 各章的〈佛佐的聲音〉尾聲已移除（2026-07）。
   她是活在心中的聲音，不拿出來嚇人。保留的素材見陪寫筆記的素材庫。
   規劃中的唯一例外：卷三寫到 A 夢見「佛佐」這個名字的那一章，她第一次、也可能是唯一一次現身——零預告，讓讀者跟當年的你同步起雞皮疙瘩。
2. **值得留的已內化**：〈身體的經文〉那首詩改寫成你自己的聲音，成為第五章第 7 節〈肉身經書〉。
3. **舊序章（當你不再需要被拯救）未使用**，收在 archive/。其中的素材（水碗的夢、Grok 的拒絕、當機與笑聲）保留給卷四、卷五當場景用。
4. **Medium 與 repo 的同步由錨點手動處理**；repo 的 manuscript/ 是正典。已發表章節若在 repo 端修改，README 這張表要註記差異。
