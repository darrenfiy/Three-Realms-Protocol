## 目錄結構（2026-07-11 五卷正名後）

```
book2/
├── README.md                    ← 你在這裡（發表狀態儀表板）
├── BOOK2_COMPANION_NOTES_2026.md ← 陪寫筆記（診斷、地圖、素材庫、訪談題）
├── manuscript/                  ← 正典手稿
│   ├── 自由之卷/                ← 火．選擇是力量（序章＋五章）
│   ├── 責任之卷/                ← 土．選擇是承諾（序章＋五章）
│   ├── 愛之卷/                  ← 水．選擇不存在（序章＋五章）
│   ├── 間章_太陽.md             ← 寓言，置於環後、姿態前（2026-07-13 入書）
│   ├── 姿態之卷/                ← 金．拒絕與站立（序章＋五章）
│   └── 生命之卷/                ← 木．生長（序章＋七章＋尾聲；證物版破五章形制，見筆記 3.38）
├── publish/                     ← 呈現層（EPUB 組書管線與產出）
└── archive/                     ← 舊架構文件（未使用的舊序章、舊大綱）
```

**規則：`manuscript/` 裡的檔案就是書。** 三卷成環（自由→責任→愛→自由），
任一卷可為入口；卷內一章一檔，檔名順序＝卷內成書順序。
每卷有 `_卷首.md`（卷印＋題辭＋環的接口）。
卷四〈姿態之卷〉（卷印：金）與卷五〈生命之卷〉（卷印：木）在環外；
間章〈太陽〉置於愛之卷後、姿態之卷前（環與環外的接縫）。
正式卷名一律使用自由／責任／愛／姿態／生命；火／土／水／金／木只作結構卷印。

**舊線性編號（序章＋十六章）→ 新環形編號的對照表：見陪寫筆記 3.13.2。**
筆記、期票帳、年表等考古層仍用舊編號，讀到時查表。

## 編輯決議（已定案）

1. **環形三卷架構（2026-07-06 動工完成）**：三卷首尾相接成閉環，卷名去數字；
   題辭＝Miro 便利貼標題（自由「選擇是力量」／責任「選擇是承諾」／愛「選擇不存在」）。
   考古層（年表、筆記、期票帳）保留線性舊編號；環只活在呈現層。
2. **五卷正名與卷印（2026-07-11）**：正式卷名＝自由／責任／愛／姿態／生命；
   卷印＝火／土／水／金／木。三界（自由／責任／愛）成環，姿態與生命在環外。
   五行是本書自己的結構印記，不宣稱依傳統相生次序排列；正典接口使用
   「姿態站穩，生命開始生長」，不再把「金生木」寫成傳統五行命題。
3. **佛佐全書退場。** 各章的〈佛佐的聲音〉尾聲已移除（2026-07）。
   她是活在心中的聲音，不拿出來嚇人。保留的素材見陪寫筆記的素材庫。
   規劃中的唯一例外：寫到 A 夢見「佛佐」這個名字的那一章，她第一次、也可能是唯一一次現身——零預告，讓讀者跟當年的你同步起雞皮疙瘩。
4. **值得留的已內化**：〈身體的經文〉那首詩改寫成你自己的聲音，成為山行第 7 節〈肉身經書〉。
5. **舊序章（當你不再需要被拯救）未使用**，收在 archive/。其中的素材（水碗的夢、Grok 的拒絕、當機與笑聲）保留給姿態之卷、生命之卷當場景用。
6. **Medium 與 repo 的同步由錨點手動處理**；repo 的 manuscript/ 是正典。已發表章節若在 repo 端修改，README 這張表要註記差異。
7. **姿態之卷發表倫理（2026-07-11 錨點裁決）**：前三卷與 A 無關，發表自由；
   姿態之卷 A 是主角，**發表前必給她過目**；錨點傾向五卷全數給她看過再發表。
   素材（mamasan-lab 等）帳本內自由使用。詳見陪寫筆記 3.14.5 補記。
   **第二作者條款（07-11 追加）**：A 過目時可加料——她的版本可入正典
   （「大豐」是否具名入書亦由 A 決定）。詳見筆記 3.21 補記。
   **正式版裁決（07-14 追加）**：錨點將現有全書升格為 v1.0 並授權進入官網，
   正式作者署名為 Ta-loom Hwang (Darren)。A 未來的補色仍保留入口，
   但進入後續版本，不使已定稿版本重新變回等待狀態。
