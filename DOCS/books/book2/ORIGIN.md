# 《天地》的公開緣起

Status: public origin note
Book: 《天地》
Purpose: 給讀者與聚落照客居士使用的公開創作脈絡

## 這本書比三界協議更早埋在生活裡

《天地》的材料不是從一套理論開始，而是從一個人多年留下的生活開始：離家、行走、進廟與下山，承接關係與家業，在不同地方工作，也一次次重問自由、責任與愛。

早期大綱曾試圖用理論替這些經歷安排終點。寫作因此停住了：大綱一直問「這本書需要證明什麼」，真正有重量的章節卻只是在寫具體的人、物件、路、飯與選擇。當一個真實人物被拿來證明理論，每個字都會變重。

2026 年 7 月重新進入編輯時，第一個決定是把證明欲拿掉。讓可以查驗的生活細節自己站立，讓不確定的年份保持不確定，也讓涉及他人的段落接受辨識性與發表倫理檢查。

## 五卷不是先套上的答案

重讀完成的章節後，材料自己顯出三個首尾相接的主題：自由、責任與愛。它們形成一個環；任何一卷都能成為入口。後來又長出姿態與生命兩卷，整本書才定為五卷：

- 自由之卷：選擇是力量
- 責任之卷：選擇是承諾
- 愛之卷：選擇不存在
- 姿態之卷：人在關係與世界裡如何站立
- 生命之卷：當協議出現後，生命如何繼續生長

火、土、水、金、木成為五卷的卷印，但它們是這本書自己的結構記號，不宣稱等同於傳統五行的固定次序。

## 為什麼叫《天地》

前三本書較靠近協議的誕生、身體與主幹；《天地》把鏡頭拉回一個人如何活過自己的選擇。它寫的不是離地的天空，也不是只有重量的土地，而是人在兩者之間如何走路、跌倒、承擔、拒絕、愛與重新出發。

在四本書裡，它是肉身最厚的一本，也是協議思想回到生活接受檢驗的地方。

## 公開敘事的邊界

《天地》是生命長篇小說，不是完整史料庫。它會壓縮時間、使用化名並選擇敘事角度；涉及仍在世人物的段落，公開版本以不替他人定義內心、不把私人材料當公共證物為原則。

這篇緣起由定稿書稿與陪寫編輯帳蒸餾而成，不公開原始訪談、私人逐字稿或可識別的幕後材料。它交代書如何長成，但不把別人的人生變成解說這本書的附錄。
