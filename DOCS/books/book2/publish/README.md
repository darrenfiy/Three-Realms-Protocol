## 呈現層決策（2026-07-13 組書、07-14 定稿）

1. **書脊順序＝線性讀者的路**：自由→責任→愛→姿態→生命。
   環的「任一卷可為入口」由閱讀地圖與接口連結實現，不動書脊。
2. **環的接口＝超連結**：只有卷首與卷尾的接口是活門（規則表見
   `build_epub.py` 的 `RING_LINKS`）；章內敘事提及維持素文。
   正典文字不增不減，連結只是讓既有的字變成門。
3. **閱讀地圖只開三扇門**（自由／責任／愛）。姿態與生命畫在環外，
   沒有門——不能跳進去，只能走出來。目次仍列全五卷（誠實）。
4. **`---` 分隔線**：後接節標題時只留空隙；場景之間渲染為置中小星記（✳）。
5. **`**副標：…**`**：視為章扉副題，去掉「副標：」標籤字，置中淡排。
6. 生命之卷尾聲「走啊」維持素文，不另加回指連結；全書閉環由文本完成，
   不讓閱讀器替最後一句指定方向。
7. 橫排。直排版（Readmoo 等）將來可用 `writing-mode` 另出一版。
