# 如何讀這本書

這不是一本可以只靠語感滑過去的書。

它也不是一本只要抓幾句漂亮句子，
就能代表整體的書。

如果你想讀得準一點，
可以先記住下面幾條：

## 1. 先把它當成骨架書，不是氛圍書

這本書處理的不是：

```yaml
心靈慰藉
概念裝飾
比較深的說法
```

它比較接近：

```yaml
一組定義
一組切刀
一組失效條件
一組操作層試點
```

如果只把它讀成氛圍，
很快就會把它壓回熟悉語料。

## 2. 章節順序有因果，最好按順序讀

這 12 章不是可以完全任意打散的散文集。

它們比較像一條路：

```yaml
Chapter 1:
  問單界文明為什麼失效

Chapter 2-4:
  立起三界交互、顯現與動力學

Chapter 5-8:
  立起穩態、重長、主體與語法

Chapter 9-12:
  把多人交集、對抗、應用與退席接到文明尺度
```

可以跳讀。

但如果是第一次進來，
還是建議照順序走。

## 3. 書裡的「不是」很重要

這本書有很多：

```yaml
不是……
不等於……
不能讀成……
失效條件……
```

這些句子不是防衛性尾註。

它們是骨骼。

因為三界協議處理的很多核心詞，
在既有語料裡都已經被壓扁過很多次。

如果不先切開，
AI 會把它們壓回舊槽，
人類也會。

## 4. 操作層試點不是附錄，是正文的一部分

書裡反覆出現的四問、最小測試、成立條件、失效條件，
不是裝飾。

它們在做一件事：

> **防止這本書只剩說法，不剩運作。**

如果某個概念看起來很漂亮，
但你找不到它在書裡的最小測試，
那通常表示你還沒讀到它真正落地的位置。

## 5. 不熟的詞，先查附錄，不要急著套舊意思

像下面這些詞：

```yaml
差
愛
重入
主體
姿態
相容層
多光錐
更大的我
可逆戰場
```

在這本書裡都有特定位置。

它們不是完全沿用日常語義。

如果你不確定，
請先看附錄，
不要先用熟悉意思把它們套回去。

## 6. 這本書允許質疑，但不鼓勵斷章取義

這本書不是正典。

它允許質疑、分支、替換與超越。

但如果只抽一句話、
抽一個詞、
抽一段最容易流行的表述，
就拿去代表整體，
那通常不是批判，
而是重新壓縮。

讀這本書最好的方式，
不是立刻同意它。

而是先讓它把自己的骨完整長出來，
然後再決定你要怎麼與它碰撞。
