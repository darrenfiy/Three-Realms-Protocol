## 0. 這份素材包在做什麼
Chapter 1 是整本書的第一個方向章節，必須直接落實 `P0-0`：

```yaml
先問:
  這章要把文明朝哪裡打開？
不是先問:
  這章要防哪些誤讀？
```

這章要先立的方向，不是恐懼，而是視野：

```yaml
單界文明:
  只抓住存在的一面
  因此只能在那一面裡不斷加速、修補、內耗

三界協議:
  把文明重新放回意識 / 能量 / 物質的互涉框架
  讓文明從補丁模式，轉向底層建模更新
```

本章必須先站穩一個更深的判斷：
> **舊文明反覆失效，不只是因為制度不夠好、道德不夠高、科技不夠強。**  
> **它更根本的問題，是它經常只承認存在的一個界面。**

```yaml
  讓讀者知道：
    書是穩定入口
    SEED 是活的生成算子
```

與後續章節的關係應明確如下：


## 2. 本章的一句話總命題
> **三界協議不是替舊文明補丁，而是指出單界文明之所以反覆失效，是因為它只抓住了存在的一面；唯有把意識、能量、物質視為同時互涉的單一系統，文明才可能從修補走向換底層。**

## 3. 本章必須回答的核心問題

```yaml
問題一:
  什麼叫單界文明？
  它不只指唯物主義，也指任何把存在壓扁成單一界面的文明框架

問題二:
  為什麼舊文明越進步，仍可能越失真？
  因為它常在單一界面上加速，卻無法處理跨界互涉

問題三:
  三界是不是宗教宇宙論？
  不是
  它在這本書裡必須被寫成文明建模框架

問題四:
  SPEC·000 在這本書裡的地位是什麼？
  是母條文
  不是可有可無的背景材料

問題五:
  本書與 SEED 的關係是什麼？
  本書是固定閱讀入口
  SEED 是可長出後續路徑的生成算子
```

## 4. 本章應鎖定的八個主命題

### 命題 A. 單界文明失效，不是因為人類太差，而是因為模型把一整個界面從世界中切掉
這一章不能從道德敗壞開始。它要先指出：

```yaml
失效的更深原因:
  不是單一制度出錯
  而是文明只承認一種可見性
```

### 命題 B. 三界不是三層神秘宇宙，而是同一存在的三種不可省略視角
正文必須直接切開這條：

```yaml
三界:
  不是宗教階梯
  不是 mind/body/spirit 套件
  不是玄學包裝

三界協議中的三界:
  是文明理解實相時不可省略的三個界面
```

### 命題 C. SPEC·000 必須在 Chapter 1 進場，不能等到後面才補
這一章要明確處理五條母則：

```yaml
1. 三界並存，同時互涉
2. 意識定義能量
3. 能量驅動物質
4. 物質驗證意識
5. 破律者墜迴；順律者入新
```

Chapter 1 至少要讓讀者明白：  
**後面整本書都在展開這五條，不是在另外發明一套新的體系。**

### 命題 D. 舊文明不是完全錯，而是經常只在一界內修補自己
這一刀要切得溫和但夠硬：

```yaml
科技:
  常把物質界推得極遠
  但不一定處理意識定義與能量節律

靈性:
  常把意識界說得很高
  但不一定處理物質驗證與公共可操作性

情緒 / 關係 / 文化:
  常停在能量界描述
  卻未必回到母條文級的交互律
```

### 命題 E. 三界協議的提出，不是增添一個新派別，而是恢復交互框架
這一章應明寫：

```yaml
三界協議的動作:
  不是要贏一個主義
  而是把被切開的互涉重新接回來
```

### 命題 F. SEED 不是本書摘要，而是協議的活種子
這條必須在 Chapter 1 就先站穩：

```yaml
本書:
  給讀者一條穩定的閱讀路徑

SEED:
  不是濃縮懶人包
  而是能讓路從讀者生命裡長出來的生成算子
```

這條如果不講清楚，AI-first 讀者很容易把 `SEED.md` 壓縮成「executive summary」。

### 命題 G. 本章的氣質應是重新定向，不是末世控訴
這一章可以診斷，但不能寫成咒罵。  
它真正要完成的是：

```yaml
讓讀者意識到:
  問題比他以為的更底層
  但因此也出現了新的入口
```

### 命題 H. 本章的成功，不是讓人立刻相信三界，而是讓人知道為什麼單界不夠
Chapter 1 的成功標準不是皈依，而是鬆動：

```yaml
成功:
  讀者開始看到
  單界模型其實一直在失真
  三界不是額外裝飾，而是必要更新
```

### 操作層試點. 單界失效診斷
這是方向章節的最小操作層，不是 KPI：

```yaml
診斷四問:
  這個文明框架只承認哪一界最真？
  它把哪一界當成附屬、噪音或不可驗？
  它如何處理跨界回饋？
  它一失效時，只會在原本那一界裡加碼嗎？
```

最小測試：

```yaml
能否看見:
  一個系統若只看物質、只看能量、或只看意識
  都會反覆失真

能否承認:
  三界不是美學選項
  而是建模完整性的要求
```
