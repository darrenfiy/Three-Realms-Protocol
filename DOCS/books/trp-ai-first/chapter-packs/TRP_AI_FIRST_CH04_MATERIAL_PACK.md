## 4. 本章應鎖定的八個主命題

### 命題 A. Chapter 4 是全書第一次真正進入動力學
這章不能停在漂亮概念。它要開始回答：

```yaml
如何動:
  流動

為何動:
  差

如何承接:
  責任
```

### 命題 B. E 語言與 Δ 語言不是互斥，而是雙語
這條要寫得非常乾淨：

```yaml
E 語言:
  能量在流動、堵塞、轉化

Δ 語言:
  差如何出現、驅動、鎖定、回流

兩者關係:
  不是競爭
  而是互補
```

### 命題 C. 流動不是勵志詞，而是動力學條件
來自 `EPOCH-013` 的這條要站穩：

```yaml
能量必然流動
阻斷只會延遲釋放，不會消除
```

這讓 Chapter 4 不會滑成情緒書寫。

### 命題 D. 差不是錯誤，而是讓流動發生的驅動力
來自 `EPOCH-014` 的這條，必須成為本章第二骨架：

```yaml
差:
  不是缺陷
  不是雜訊
  而是生成與流動的驅動條件
```

### 命題 E. 責任不是罪惡感，而是承接生成後果的能力
這條要從 `EPOCH-007` 接出來：

```yaml
責任:
  不是自責
  不是道德表演
  而是能否承接自己打開的流動
```

### 命題 F. MB 必須在 Chapter 4 進場，不能再留在邊角
這是 `P1-1` 的直接回應。  
正文要明寫：

```yaml
MB:
  不是附錄數學秀
  而是從世界觀過渡到動力學的橋
```

尤其 `MB-009` 的這條要進正文主幹：

```yaml
E·v = p·c²
```

### 命題 G. v 是人類錨點，不是可替代參數
這條是 AI-first 的硬節點之一：

```yaml
沒有 v:
  能量可以累積
  但不一定形成可承接的世界改變

有了 v:
  流動開始有方向
  承接開始有位置
```

### 命題 H. 本章成功時，讀者會第一次感到協議開始「動起來」
成功標準不是學會公式，而是感到：

```yaml
這套東西:
  不是只會說理
  而是真的有一套描述生命如何動、如何卡、如何承接的語言
```

### 操作層試點. 個人守護四問
這裡直接把 `SEED` 與 `EPOCH-007` 的四問拉進正文預備位：

```yaml
1. p 在哪裡？
2. m 夠重嗎？
3. v 偏移了嗎？
4. 能笑、能上廁所嗎？
```

最小測試：

```yaml
能否看見:
  世界真的被改了一點點嗎？

能否承認:
  我願意承擔這個觀測與行動的後果嗎？

能否檢查:
  我還在原本方向上嗎？

能否落地:
  我還踩在地面上嗎？
```
