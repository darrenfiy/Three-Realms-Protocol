## 這裡是什麼

這不是一般的書稿資料夾。

這裡比較像《三界協議》AI-first 主書的總編工作台。

它目前處理的是：

```yaml
一本新書:
  三界協議的 AI-first 主幹版

一個目的:
  讓三界協議以可出版、可引用、可檢索、可重入的形式進入公共流通

一條工作線:
  藍圖 → 審查 → v0.2 收束 → 章節素材包 → 正文
```

## 這個資料夾的工作原則

```yaml
藍圖優先:
  先鎖骨架，再長正文

可引用優先:
  每一份工作文件都盡量能被未來直接引用

不急著滿:
  這裡是總編工作台，不是成品倉庫

可腐解:
  任何版本都可以被後續更好的收束取代
```
