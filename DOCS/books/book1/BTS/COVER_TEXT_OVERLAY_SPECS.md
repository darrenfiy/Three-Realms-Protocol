## ✍️ 文字疊加規格

### 主標題「呼吸」

#### 位置
- **垂直位置**：圖像中心偏下區域（約在畫布高度的 62-65% 處）
- **水平位置**：完全居中
- **避開區域**：兩個人形輪廓、中央明亮的能量流

#### 最佳放置點
觀察圖像，文字應放在：
- **下方深色區域**（紫藍色漸層處）
- 兩個人形下方、畫布底部上方
- 光芒不會過度干擾文字的位置

#### 字體
**推薦字體**（按優先順序）：
1. **思源黑體 Heavy/Bold**（Noto Sans CJK TC Heavy）
2. **蘋方-繁 粗體**（PingFang TC Semibold）
3. **黑體-繁 中黑**（Heiti TC Medium）

**特性要求**：
- 粗體，清晰有力
- 現代、簡潔
- 在複雜背景上仍清楚易讀

#### 字體大小
- **建議**：圖像高度的 7-9%
- 如果圖像是 2560px 高度，約 **180-230px** 字體大小
- 調整原則：在縮圖（200px寬）下仍清晰可讀

#### 顏色方案

**方案一（推薦）：溫暖白色**
```
主色：溫暖白
- RGB: (255, 250, 245)
- Hex: #FFFAF5
- 不透明度: 95-100%

描邊（可選）：
- 顏色: 深紫藍色 #1A0F3A
- 寬度: 2-3px
- 讓文字在亮處和暗處都清晰
```

**方案二：金白漸層**
```
漸層方向：從左到右或從上到下
起始色: #FFE4B5 (柔和金)
結束色: #FFFAF5 (溫暖白)
不透明度: 95-100%
```

#### 視覺效果

**建議效果**：
- **微妙的光暈**（非常輕柔，不要霓虹燈效果）
  - 光暈顏色：金白色 #FFF8DC
  - 模糊半徑：8-12px
  - 不透明度：30-40%

**不要**：
- 硬陰影（drop shadow）
- 過度光芒（outer glow太強）
- 3D效果
- 漸層太明顯

#### 字間距
- 兩個字之間：微妙拉開（約字寬的 10-15%）
- 不要太擠，給每個字呼吸空間

### 副標題（作者署名）

#### 文字
```
Darren Hwang & 協議身體
```

或者更簡潔：
```
協議身體 著
```

#### 位置
- 在主標題「呼吸」正下方
- 間距：約主標題字高的 40-50%
- 完全居中對齊

#### 字體
- 同主標題系列，但使用 **Light 或 Regular** 字重
- 或混合：
  - 英文部分：Helvetica Neue Light / Avenir Next Regular
  - 中文部分：思源黑體 Regular / 蘋方 Regular

#### 字體大小
- 約主標題的 **18-22%**
- 如主標題 200px，副標題約 36-44px

#### 顏色
```
淺灰帶金色調
- RGB: (200, 195, 190)
- Hex: #C8C3BE
- 不透明度: 85-90%
```

## 📐 技術規格檢查

### 最終輸出要求
- **尺寸**：1600 x 2560 px（Amazon KDP標準）
- **解析度**：300 DPI
- **格式**：
  - PNG（主要格式，支援透明）
  - JPEG（備用，給不支援PNG的平台）
- **色彩模式**：RGB
- **檔案大小**：< 5 MB

### 確認檢查清單
- [ ] 原圖 breathing_cover.jpg 維持原始視覺完整性
- [ ] 「呼吸」二字清晰、醒目、易讀
- [ ] 縮小到縮圖尺寸（200px寬）仍可辨識
- [ ] 文字不會覆蓋或破壞圖像的關鍵視覺元素
- [ ] 作者署名清楚但不搶主標題風采
- [ ] 整體和諧，像是一體的設計

## 🛠️ 實作建議

### 工具選項

**選項一：Photoshop / GIMP**
1. 開啟 breathing_cover.jpg
2. 確認解析度 300 DPI、尺寸 1600x2560px
3. 新增文字圖層
4. 套用字體、大小、顏色
5. 加入光暈效果（外光暈，溫柔模式）
6. 儲存為 PNG 和 JPEG

**選項二：Canva**
1. 上傳 breathing_cover.jpg 作為背景
2. 設定畫布 1600x2560px
3. 添加文字元素
4. 調整字體、位置、效果
5. 下載為 PNG（300 DPI）

**選項三：Python PIL (Pillow)**
```python
from PIL import Image, ImageDraw, ImageFont, ImageFilter

# 開啟基礎圖像
img = Image.open('breathing_cover.jpg')

# 確保尺寸
img = img.resize((1600, 2560), Image.LANCZOS)

# 創建文字圖層...
# （需要詳細程式碼時再補充）
```

**選項四：AI輔助（Claude Desktop / ChatGPT）**
- 上傳 breathing_cover.jpg
- 提供本規格文件
- 請求添加文字
- 可能需要多次迭代調整
