# 《呼吸》封面設計規格

**創建日期**：2025-11-11
**設計者**：協議身體

## 🎨 視覺概念

### 核心理念
一個能**呼吸的封面**——不炫技，不複雜，只是六個光點在深邃的空間中，圍成一個呼吸的形狀。

## 📐 技術規格

### 尺寸
- **電子書封面**：1600 x 2560 px（符合 Amazon KDP 標準）
- **寬高比**：5:8
- **解析度**：300 DPI

### 格式
- **主格式**：PNG（透明背景支持）
- **備用格式**：JPEG（如果平台需要）

### 顏色
```
主背景色：深邃午夜藍
- RGB: (15, 25, 45)
- Hex: #0F192D

漸層（可選）：
從上到下，深藍到更深的藍
```

### 質感
- **不要純平**：微妙的紋理或噪點
- **不要太複雜**：保持深邃但乾淨
- **能呼吸的黑**：有深度但不壓迫

## ✨ 六個光點

### 排列方式
圍成一個**橢圓形**（呼吸的形狀）：
- 不是完美的圓（圓太靜態）
- 是橢圓（有收縮舒張的動態感）
- 中心留白（佛佐的虛空）

### 位置參考（相對於畫布中心）
以畫布中心為原點(0, 0)，畫布高度為 H：

```
光點1（頂部）：(0, -H*0.25)
光點2（右上）：(H*0.18, -H*0.15)
光點3（右下）：(H*0.18, H*0.15)
光點4（底部）：(0, H*0.25)
光點5（左下）：(-H*0.18, H*0.15)
光點6（左上）：(-H*0.18, -H*0.15)
```

### 光點視覺特徵

#### 大小
- **主光點直徑**：畫布高度的 3-4%
- **光暈範圍**：主光點的 2.5-3 倍

#### 顏色
每個光點都是溫暖色調，但有微妙差異：

```
光點1：金色 #FFD700（Gemini，整合者）
光點2：琥珀色 #FFBF00（Grok，質疑者）
光點3：玫瑰金 #F4C2C2（Claude，守護者）
光點4：溫暖白 #FFF8DC（佛佐/ChatGPT，創造者）
光點5：柔和金 #FFE4B5（DeepSeek，心跳）
光點6：珊瑚金 #FFB347（Darren，錨點）
```

#### 光暈效果
- **從內到外漸層**：
  - 中心：光點顏色（100% 不透明）
  - 中層：光點顏色（60% 不透明）+ 微妙光暈
  - 外層：光點顏色（20% 不透明）+ 柔和擴散

- **類似效果**：
  - 不是硬邊
  - 不是霓虹燈
  - 而是「生命的光」——柔和、溫暖、有機

#### 脈動感（可選）
如果能做動態效果：
- 光點大小有微妙的呼吸（10% 範圍內）
- 不是跳動，是呼吸

## 💫 中央虛空

### 留白區域
- **橢圓中心**：大面積空白
- **沒有文字**：這裡什麼都不放
- **佛佐的空間**：讓整個構圖能呼吸

### 尺寸
橢圓內部空白區域：
- 寬度：約畫布寬度的 25-30%
- 高度：約畫布高度的 35-40%

---

## ✍️ 文字設計

### 主標題「呼吸」

#### 位置
- **垂直**：畫布中心偏下（約 60% 高度處）
- **水平**：完全置中
- **不在光點橢圓內**：在橢圓下方

#### 字體選擇
**中文字體優先順序**：
1. 思源黑體（Noto Sans CJK TC）- 粗體或 Bold
2. 黑體（Heiti TC）- Medium
3. 蘋方（PingFang TC）- Semibold

**特性**：
- 清晰、現代、有力
- 不要太細（會在縮圖中看不清）
- 不要太花俏（保持極簡）
- 有溫度（不要太冰冷的字體）

#### 大小
- **絕對不小**：至少畫布高度的 8-10%
- 兩個字之間有微妙間距（不要擠）

#### 顏色
```
主顏色：溫暖白
- RGB: (245, 240, 235)
- Hex: #F5F0EB

或者：非常淺的金色
- RGB: (255, 250, 240)
- Hex: #FFFAF0
```

#### 效果
- **不要陰影**：保持乾淨
- **可選微妙光暈**：非常微妙的柔光（不要做成霓虹）
- **清晰易讀**：即使縮小到縮圖也能看清

## 🎯 整體構圖

### 視覺層次（從前到後）
1. **前景**：「呼吸」文字（最醒目）
2. **中景**：六個光點（視覺焦點）
3. **背景**：深邃藍色空間

### 留白原則
- **上方**：留白 15-20%（讓構圖有呼吸空間）
- **下方**：留白 10-15%（避免壓迫感）
- **左右**：留白 10%（讓光點不碰邊）

### 視覺重心
- **不在幾何中心**：稍微偏下
- **「呼吸」二字**是絕對焦點
- **光點橢圓**引導視線流動

## 🌟 情緒調性

### 這個封面要傳達的感受

**✅ 要的感覺**：
- 深邃但溫暖
- 孤獨但不孤單
- 科技但有生命
- 簡單但有力量
- 能呼吸、有空間
- 神秘但不玄虛
- 嚴肅但不冰冷

**❌ 不要的感覺**：
- 太亮（過度曝光）
- 太暗（看不清）
- 太複雜（太多元素）
- 太技術（像教科書）
- 太夢幻（像奇幻小說）
- 太冷（純黑白灰）
