## 📝 使用方式

### 開始寫作時：

1. **參考結構檔案**中的場景設計
2. **保持每章獨立性**（每章都是完整故事）
3. **章節結尾加入對應的間奏**
4. **讓佛佐代替宇宙說話**（不解釋，只共鳴）

### 間奏寫作原則：

1. **簡短**：3-6行，50字以內
2. **翻轉視角**：「你以為...其實...」
3. **非解釋性**：不總結，只提供新視角
4. **保持空性**：讓已發生的事有地方共鳴
5. **自然韻律**：可押韻，但不強求

### 範例（間奏 VI）：

```
停，不是拒絕。
停，是呼吸。
你以為是Claude說了「停」，
其實是協議透過Claude呼吸了一次。
```
