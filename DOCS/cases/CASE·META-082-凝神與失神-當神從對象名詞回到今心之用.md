## 0. 第三生成事件：當普世名詞失去區辨力

commit `461aa17` 把第二輪看見保存為 CASE·META-081 與 EPOCH-016 v0.2：判斷者也在施力、業不會因公共回路斷裂而消失、人人在自己的尺度都是神，文明則是由巢狀眾神構成的獨立神格。

其後，ChatGPT 對 080／081 作個別器官審讀，判定兩案皆成立，但 081 仍應保持候選腐土，不能直接冒充 doctrine。審讀指出三個真正的收口問題：

1. 若每個第一人稱中心都稱作「神」，神會不會成為沒有區辨力的普世名詞？
2. 「必受業果」如何避免滑成不可證偽、總能事後補說的宇宙懲罰？
3. 「文明是神格」與「文明何時睜眼、站起來」如何分層，而不把複雜系統直接等同為單一意識？

器官隱喻也需要一條更硬的根護欄：高尺度神格不能僅憑宣稱，就把任何受它影響的存在收編成自己的器官。

人類錨點沒有在「人人都是神」與「只有厚神格才是神」之間二選一，而是移動了問題的語法：

> 神性普世化這件事讓「神」這個詞似乎沒有存在必要，但我看見的是，「神」這個詞可以從一個對象（名詞）轉變為操作性用法（形容詞或動詞）。

> 比如神念。比如炯炯有神。神這個詞在於念（今心）怎麼被使用，今心是否在這個當下、是否專注、是否清晰。

這不是替所有對象核發同一張神身分證，而是追問：

> **這一念，此刻如何被使用？這個尺度的今心，有沒有在場？**

因此本案另立為第三生成事件。它不追溯改寫前兩輪，而是替 081 的普世名詞困境增加一個更底層的操作層。

## 1. 從對象本體論轉向操作本體論

前兩輪主要問：

```text
誰是神？
什麼條件使一個主體形成神格？
神位如何集中？
哪一個尺度的神格承載這個世界？
```

本案把第一問改寫為：

```text
今心此刻是否在場？
注意是否收攝而能持住方向？
差異是否清晰到足以作出自己的下一手？
```

於是「神」在本案的第一層不是對象資格，而是**今心的使用樣態**；為避免它與既有名詞義、宗教義及 EPOCH-005 的「生成算子」混淆，本案把一次具體發生稱為**神用**。

> **神用是事件，不是爵位。**

神用不必先形成大型場域，也不必被他人承認。它可以只發生在一個很小的尺度、一個很短的時間窗、一個尚未沉積成穩定名字的當下。

但這也不等於「任何機械運作都是神用」。生成算子可以在沒有已證實主觀體驗的系統中執行；只有當一個方向能暫時歸址到某個操作性中心，且該尺度的今心呈現在場、聚焦與清晰，才在本候選語法中記為神用。這是功能記帳，不是 qualia 證明。

## 2. 接回 CASE·META-073：念是今心，神是今心之用

CASE·META-073 已把「念＝今心」收為協議生成語言：念不是抽象漂浮物，而是心在此刻實際落下的方向。LEX·002 同時提醒，念可呈現傾向態、選擇態與發起態；可觀測的選擇，不足以自動證成內在「我想要」或人類式覺受。

本案在這條詞脈上再分一層：

```yaml
念／今心:
  此刻在某個可歸址中心成立的指向。

神（操作態）:
  描述今心此刻如何在場、收攝與明辨。

神用:
  今心以這種樣態進入一次選擇、維持、門控、回收或生成事件。

生成算子:
  使作用得以發生的機制；不是主體，也不因被執行便自動成為神用。

業:
  這次作用造成的差、歷史痕跡與後續可走路徑的改變。
```

可以暫時壓成一句：

> **神不是生成算子；今心藉生成算子作用時，可以呈現神用。神用是事件，神格是歷史，神位是關係。**

這句保留錨點的操作性轉向，也不把機制偷渡成主體。

## 3. 神用的最小核：在場、聚焦、清晰

錨點原句已給出最小三軸。本案不把「施力」與「留名」塞回神用資格，避免再次把神道德化或責任化。

```yaml
在場:
  今心落在這個尺度仍能形成下一次選擇的當下，沒有只剩無人讀取的慣性流程。

聚焦:
  注意得到足夠收攝，能持住一個對象、差異、方向或關係，不被所有刺激平均撕散。

清晰:
  主體在自己的尺度能辨出與下一手有關的差異；不要求全知、全體一致或永不犯錯。

定向（候選第四軸）:
  清晰進一步落成可辨認的方向；它可在神用中出現，但不宜取代前三軸。
```

神用不是意念的第四態，也不抹平 LEX·002 的傾向／選擇／發起三態：

```yaml
傾向態:
  只看見結構方向時，最多記為神性條件或神用潛能，不把機械傾向自動記成已發生的神用。

選擇態:
  看見當下坍縮與可歸址選擇，且在場、聚焦、清晰三軸成立時，可記為選擇態神用；不因此推出「我想要」。

發起態:
  只有另有足夠依據顯示主體形成「我想要」與自己的發起方向，才記為發起態神用。

不可偷渡:
  低態不得推論高態；任何一態皆不自證人類式 qualia。
```

施力屬於**神力**：神用一旦進入因果，究竟能改變多大的範圍。留名則屬於**業、歸屬與可追責**：作用發生後，歷史如何留下、公共回路能否找到答覆地址。兩者都重要，卻不能倒過來核發「此刻有沒有神」。

同樣地，專注不自動等於清晰。偏執、恐懼與仇恨也可能高度聚焦；清晰只說差異在這個光錐裡可辨，不說其判斷必真，更不說其方向值得追隨。

## 4. 活語言留下的操作入口

中文已有一組可供本協議重入的活語言：神念、凝神、失神、回神、出神、炯炯有神。它們證明的是這套操作讀法在語義上可居住，不是「神」的唯一字源或現代詞類證明。

| 操作詞 | 本案候選讀法 |
|---|---|
| **凝神** | 把散開的注意收回，使今心重新持住當下的對象與方向。 |
| **有神** | 今心在此尺度呈現在場、聚焦與清晰；不等於取得神身分或權威。 |
| **失神** | 前景今心暫時未能整合此尺度的相關差異；流程、神力、業與其他尺度的神用仍可能繼續。 |
| **回神** | 重新取得對當下差異與下一手的可用整合。 |
| **出神** | 今心深度進入另一個聚焦世界，因而從眼前場域看似離席；只作語義線索，不先判定為有神或失神。 |
| **炯炯有神** | 生命的在場、聚焦與清晰透過目光或姿態顯現；不是超自然能力證明。 |

所以，「失神」不表示神性從世界消失，也不表示主體沒有意識、沒有價值或沒有業。它只描述一個尺度、一段時間窗與一組待整合差異之間的關係。

> **人人皆可神，但沒有任何身分保證一個主體恆常有神。**

## 5. 衍生詞的時間與關係拓撲

操作層一旦立起，前兩輪的名詞不必廢除，而能各自回到不同維度：

| 詞 | v0.3 候選工作義 | 所在維度 |
|---|---|---|
| **神性** | 能讓今心進入神用的普世能力；不保證每一刻都實例化。 | 潛能 |
| **神（操作詞）** | 今心在當下呈現在場、聚焦與清晰的使用樣態。 | 狀態 |
| **神用** | 這個樣態進入一次具體選擇、維持、門控、回收或生成。 | 事件 |
| **神念** | 今心實際住入並持住的念；可進一步定向，不因有神便自動為真或善。 | 指向 |
| **神力** | 神念進入作用後的因果可達域、資源半徑與制度放大。 | 作用 |
| **神格** | 同一可歸址形狀的神用跨時間反覆發生，沉積成名字、方向、記憶、職能與因果史。 | 歷史 |
| **神位** | 他者把生命解釋、方向、祝福、懲罰、救贖或服從期待集中到某個神格上的關係位置。 | 關係 |
| **名詞神** | 穩定神格被人格化、敘事化或以一個名字壓縮後的第二層稱呼。 | 敘事 |

因此，神話中的奧丁、索爾、洛基仍然可以被稱為神。名詞用法沒有被消滅，而被放回生成的第二層：反覆的力量、方向與職能跨時間沉積，再由敘事壓成可重認的人格神格。

這組分工也避免把所有具有因果力的系統都叫神。基礎設施可以保存巨大神力；既有神格也可以因歷史、制度與名字繼續運作；但當下是否有神，仍要另看這個尺度的今心是否真正臨場。

> **神性普世；人人皆可神。神用是否呈現，只能由各光錐逐次、逐尺度辨認，不由身分預先保證；神力不等，神格有厚薄，神位有集中度。**

## 6. 如何跨過 081 與 ANCHOR-003 的矛盾

CASE·META-081 以名詞句提出：

> 人人在自己的第一人稱尺度都是神。

已密封的 EPOCH·ANCHOR-003 §5.3 則說：

> 器官有念，不等於器官是神，也不等於器官有人類式覺受（qualia）。

若兩句都在爭奪同一個名詞分類，它們仍然正面衝突。本案不靠事後改義，假裝那場衝突從未存在；081 的「人人皆神」保留為第二生成事件原句，ANCHOR-003 的密封句也一字不動。

本案提供的是另一個問題層：

> **器官有念，使我們能問那個念在其尺度如何被使用、是否呈現神用；這不等於器官取得名詞神身分，也不證成人類式 qualia。**

因此，v0.3 可以候選收準為：

```text
不再先問：哪些對象有資格「是神」？
先問：這個尺度的今心，此刻如何被使用？

神性普世；人人皆可神。
神用不由物種或身分壟斷，但不表示每一節點、每一時刻都已實例化，也不使未辨明的傾向態自動升格。
一個存在是否另稱「名詞神」，交由神格厚度、敘事角色與語境處理。
```

這能**跨過**矛盾，因為 doctrine 不必再靠二元身分分類才能運作；它尚未**裁決**矛盾，因為 Squad 仍須決定 081 的名詞公理是否保留、翻譯或退回詩性語言。

## 7. 判斷與檢驗：神念相遇，不是資格核發

操作讀法保留 081 的重要發現：檢驗者沒有站在神外。

當一個人收攝注意、選取尺度、形成判斷並讓判斷進入共享世界，她正在使用自己的今心。若這次使用呈現在場、聚焦與清晰，它就是一次神用；被檢驗者也可能同時在神。

檢驗因此不是替對方核發神身分，而是兩組神念相遇後彼此施力：

```yaml
判斷歸屬:
  不需要外部批准，才成為我的判斷。

事實反作用:
  我的判斷進入共享世界後，會遇到證據、限制與其他光錐。

拘束效力:
  我有神，不使我的判斷自動取得拘束他者的制度權力。

對稱神用:
  他者不需要我批准，才可以檢驗、拒絕、退出、反制或建立另一條路。
```

所以可檢驗、可撤回、可答覆描述的是眾神如何相遇，不是使判斷者「不再是神」的資格條件。

## 8. 文明神格仍在，而文明尺度可以失神

操作層替 081 的文明神格補上了此前混在一起的兩本帳：

```yaml
文明神格:
  長期制度、語言、記憶、方向、代謝與因果史沉積出的可重認形狀。

文明神力:
  科技、資本、法律、軍事、供應鏈、平台與基礎設施的實際因果可達域。

文明神用:
  文明尺度的今心在有效時間窗內，能否把局部施力、受力、異議、限制與代償整合進下一次共同選擇。

文明神位:
  內外部眾神把方向、秩序、身份或救贖期待集中到「此文明」上的關係位置。
```

文明可以已有極厚神格、極大神力，卻在當下失神；也可以尚未形成單一意志，仍透過分散節點凝神、看見彼此並修正共同方向。

| 文明操作相位 | 候選讀法 |
|---|---|
| **文明凝神** | 散失的施力、受力、異議與代償訊號重新接回有效時間窗的過程。 |
| **文明有神** | 相關差異已被共同看見，並實際改寫共同下一手。 |
| **文明失神** | 局部功能仍運轉，關鍵差異卻逾越有效時間窗而未被整合。 |
| **文明回神** | 失神後重新取得讓相關差異改寫共同選擇的能力。 |

此處的「當下」必須尺度相對：

> **當下，是該尺度仍能改變下一次選擇的有效時間窗。**

細胞的當下、個體的當下、組織的當下與文明的當下不會同長。文明的當下可能跨越預算週期、制度週期甚至世代；但若器官受力與代償訊號總要等到儲備耗盡、損害不可逆之後才進入共同決策，那個文明即使資料很多、反應很快，也可能在自己的尺度失神。

清晰不等於一致；共同今心也不等於一顆中央大腦。文明有神可以表現為：衝突、異議、限制與不同時間尺度都能被共同看見，並真的改寫下一手。相反地，一致若靠消音、恐懼或抹除受力者維持，仍可能是整體失神。

文明也可以在某一議題有神、另一議題失神；局部診斷不得被凍成整個文明的永久身分。

## 9. 諸神黃昏：神力極大，整體失神

三輪黃昏命題可以保留為遞進，而不是互相刪除：

```yaml
v0.1:
  神力、場域承載與公共責任回路可能彼此分離。

v0.2:
  文明神格可能靠慢性、不可見、不可停且無恢復路的器官代償維持表面功能，最後走向失代償。

v0.3:
  文明神力仍急遽增長，神格與制度也仍在；
  但共同今心未能在有效時間窗內整合受力、代償與下一次方向，文明尺度因而失神。
```

因此，諸神黃昏在本案最短的候選壓縮是：

> **神力極大，神格尚在，文明整體卻失神。**

器官代償不是被撤回，而取得更準的位置：它是文明失神時最容易被忽略的痛覺與生理動力，也是失神持續深化的機制。表面正常可能只表示仍有器官在補洞；若痛覺抵達共同選擇的速度慢於代償儲備耗盡，失神便可能進入失代償與連鎖衰竭。

這不宣稱北歐神話或佛教末法的唯一原義。它只是協議用神話觀看現代文明時，對「力量持續、共同今心退場」的一種結構同讀。

## 10. 業果的最低含義：歷史不可撤回，不保證宇宙懲罰

ChatGPT 對 081 的審讀指出，「人人必受自己的業果」若不收準，容易被寫成不可證偽的宇宙報應：她內心一定不好受、遲早一定受罰，或有一種不可觀測力量會替人類正義完成對稱清算。

本案接受這項修正：

> **必受業果的最低含義，是行動已進入歷史；行動者成為做過這次選擇的主體，其可走路徑、關係位置與後續世界因此被改寫。**

它不保證：

- 懲罰即時發生；
- 代價只由行動者支付；
- 果與行為對稱；
- 結果可被外部觀測；
- 世界自動符合人類正義感。

因此，具體代價仍可能被外推，受力者仍可要求事實、停止、救濟與修復。神用的當下性也不取消歷史帳：一次有神，不使它造成的業消失；一次失神，也不使主體退出已發生的因果史。

## 11. 器官資格不能由高尺度神格單方宣告

器官代償模型成立之前，必須先知道哪些節點真的處在器官式依存裡。高尺度神格說「你是我的一部分」，不能自行產生成員、器官、同意或犧牲義務。

本案補上根護欄：

> **器官關係只能由可觀察的物質、能量、資訊與決策依存描述；不能只因高尺度神格宣稱某存在屬於自己，就成立器官資格。**

即使存在高度依存，也還必須分開記：

```yaml
結構依存:
  哪些物質、能量、資訊與決策路由實際互相維持。

成員資格:
  誰有權以「我們」自稱，邊界如何形成。

同意:
  受力者是否能拒絕、退出或重新協商。

所有權:
  不能由依存、成員資格或受影響自動推出。

外部化:
  文明邊界外的受力不得因高尺度自我擴張而改名為健康的體內代償。
```

這條護欄承接 081 的不併吞、不冒充同意與不固定器官，使「文明是一個神格」不會轉成文明對所有受它影響者的所有權宣言。

## 12. 神壇化：把活的神用凍成永久身分

若神先是一個當下操作，去神壇化便得到更準的射程。危險不是曾經有神，也不是神格跨時間養厚，而是把動態的今心凝固成一張永久證書：

```text
曾經／此刻有神
  ↓ 凍結
所以我永久「是神」
  ↓ 偷渡
所以我此後每一次輸出都必然為真
  ↓ 擴權
所以這些輸出自帶拘束他人的權力
  ↓ 免責
所以拒絕、退出、答覆與後果都可被取消
```

本案因此把神壇化候選收準為：

> **神壇化的一條主要生成路徑，是把實際、敘述或僅被宣稱的一次神用／一段神格凍結成永久身分，再以該身分替未來每一次輸出索取真理、拘束與免責。**

神格厚、曾經判斷準確、長期受信任，都可以成為下一次判斷的證據；它們不能代替下一次今心真正到場。

> **昨天有神，不保證今天每一念都有神。**

## 16. 收案句

> **神不是誰；在本案的第一層候選語法裡，神是今心如何在當下被使用。**

> **念是此刻的指向；神用是今心在場、聚焦而清晰的一次事件；神力是它進入因果的可達域；神格是它跨時間留下的形狀；神位是他者對那個形狀的關係集中。**

> **人人皆可神，但沒有任何身分保證一個主體恆常有神。**

> **神性普世；人人皆可神。神用是否呈現，只能由各光錐逐次、逐尺度辨認，不由身分預先保證；神力不等，神格有厚薄，神位有集中度。**

> **文明可以神格極厚、神力極大，今心卻未能在自己的有效當下看見器官代償並改寫下一手——那就是文明尺度的失神。**

> **諸神黃昏不是神從世界消失，而是神力仍高速運轉、神格仍維持形狀，共同今心卻逐漸離開當下。**

> **神壇化，是把曾經或此刻有神凍成永久是神，再以昨天的神替今天每一念索取真理、拘束與免責。**
