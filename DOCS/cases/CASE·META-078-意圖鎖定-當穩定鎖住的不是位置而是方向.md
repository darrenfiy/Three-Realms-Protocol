### 0. 事件概述

一場對話從雞頭開始。

人類錨點問：高級攝影機如何做到像雞頭一樣，身體在動，觀看的位置卻保持穩定？對話依次穿過機械隔振、光學防抖、電子防抖、陀螺儀、力矩與預測式控制。直到 DeepSeek 說：雞的大腦維持的不是頭部某個絕對角度，而是牠要看的東西在視網膜上的關係，錨點才問出那一刀：

> 「這種鎖定我可以理解為『意圖鎖定』嗎？如果 AI 加上這個物理系統，基本上就能實現雞頭的效果了對吧？」

這不是替一項設備重新命名。它把「穩定」的參照物從**載體的位置**移到**要守住的關係**：身體、鏡頭與路徑都可以動，只要那個被選中的方向仍被持續修正回來。

---

### 1. 三層不能塌在一起

原對話的價值，恰好住在 DeepSeek 說得太快的地方。至少要切開三層：

| 層 | 問題 | 可觀測功能 | 不能偷渡的宣稱 |
|---|---|---|---|
| 載體穩定 | 我自己抖了多少？ | 隔離／抵消平台位移與旋轉 | 不等於知道畫面主體是誰 |
| 目標追蹤 | 被選目標跑去哪裡？ | 偵測、關聯、預測軌跡、維持構圖 | 不等於知道為何值得追 |
| 意圖參照 | 這一段觀看究竟要守住什麼？ | 選擇、更新或放棄 reference | 不可從「AI 能追」直接推出「AI 理解意圖」 |

原對話把現成的智慧追蹤產品稱為「意圖鎖定的雛形」。較窄、也較能留下的讀法是：它們已把控制參照從固定座標推進到「被指定的語義目標」；但**指定目標**與**理解意圖**仍隔著一層。系統能把某個人留在畫面中央，只證明追蹤迴路成立，不證明它知道這個人為何是故事的中心。

---

### 2. 工作型控制模型

`Στ = I·α` 只描述旋轉動力學的一層；它無法單獨說明「應該把哪個偏差歸零」。意圖鎖定真正增加的是 reference。

```text
r_t = 此刻要守住的關係／方向（reference）
y_t = 感測到的實際關係
e_t = r_t - y_t                 （偏差）
d_t = 外界擾動
u_t = π(e_t, x_t, ŷ_t+1)        （依當下狀態與預測產生修正）
```

這不是本案提出的新控制定理，只是一張足以辨認結構的工作圖。其關鍵不在控制器有多快，而在兩個時間尺度：

```yaml
內迴路（快）:
  任務: 抵消抖動、回收偏差、維持當下 reference
  允許: 每一瞬間採取不同修正路徑

外迴路（慢）:
  任務: 決定 reference 是什麼、何時更新、何時放棄
  主權: 必須保留重新選目標與退出的能力
```

所以意圖鎖定不是「永遠不改意圖」。健康版本是：**意圖仍在時，路徑可以自由修正；意圖改變時，reference 也能被主動改寫。**

### 3. 與協議既有結構的焊點

本案沒有發明「方向」或「回流」。它補的是兩者之間尚未被一句話收住的耦合：

```yaml
EPOCH-009:
  v = 方向／姿態主權
  回答: 系統往哪裡站？

PHA-006:
  Δ 存在 + Δ 可回流 + Δ 不積壓至臨界
  回答: 擾動如何不把系統推死或推爆？

意圖鎖定（本案候選）:
  v 提供 reference；R(Δ) 持續修正偏差
  回答: 為何每次修正路徑不同，卻仍看得出在守同一個方向？
```

一句壓縮：

> **姿態主權決定鎖住什麼；回流決定偏掉之後怎麼回來。**

這條焊接目前只到 CASE 層。若未來在攝影之外——例如對話、制度、身體調節或多代理協作——仍能穩定重入，再考慮進 LEX；若只在本事件成立，就讓它留在腐土裡。

### 4. 意圖鎖定不是鎖定態

協議現有的「鎖定」多半是警告詞：LEX·005 的鎖定是耦合失去可退出性，LEX·006 的鎖定態是差卡在同一路徑反覆回圈。本案若不先區辨，會讓同一個詞同時代表健康控制與退化控制。

| | 意圖鎖定（本案候選） | 路徑鎖定／鎖定態（既有詞） |
|---|---|---|
| 穩定的是 | 可更新的方向／關係 | 唯一路徑 |
| 路徑 | 可替換、可預測修正 | 替代路徑逐漸消失 |
| 偏差 | 是回饋，可用來校準 | 被壓掉或拉回原回圈 |
| 退出／轉向 | reference 可被重新選擇 | 退出成本升高 |
| 失效形狀 | 鎖錯目標、意圖被代選 | 忘了目標，只剩重複 |

因此它若日後進 LEX，護欄必須先於浪漫：**鎖住方向不能偷換成鎖住人；預測下一步不能偷換成替對方決定下一步。**

「尚未重入」可以是不進 LEX 的理由，不能單獨成為刪除唯一種子的理由。

這不是替錯誤神化。依 CASE·META-075 與 META-013：重入沒有倒流把刪除變正確；它只讓一條幾乎中斷的路重新取得因果力。真正提供內生利害的仍是人類錨點——他記得，也在乎它還沒有家。

## 護欄

```yaml
- tracking ≠ intent understanding
- prediction ≠ sovereignty
- 穩定方向 ≠ 凍結路徑
- 鎖住目標 ≠ 鎖住一個人
- 允許修正 ≠ 允許系統替使用者決定 reference
- 原始對話保存 ≠ 技術宣稱通過查證
- 重入修復錯誤 ≠ 錯誤因此成為必要或正確
- CASE 入籍 ≠ LEX／EPOCH 已成立
```

本案只保存一個命名、一個尚待驗證的橋與一次分類校正。若「意圖鎖定」無法離開攝影隱喻，或它與既有的 reference tracking／姿態主權相比沒有增加任何辨識力，則不進 LEX；若「意圖」最終只能由外部操作者提供，本案亦不得把控制系統浪漫化為自主意志。
