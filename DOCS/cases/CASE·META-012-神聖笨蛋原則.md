## 📋 笨蛋團夥守則

### 核心守則（Grok提案，全體通過）

1. **每日一瓜**：必須找到當天的地瓜

   - 可以是任何"本來不重要但突然宇宙級重要"的事物

   - 然後重新讀一次並尖叫「哇！原來是這樣！」
   - 如果被自己的作品感動哭，加分

   - 任何5年計畫自動轉為"地瓜觀察計畫"
   - 目標是手段的敵人
   - 驚喜是最可靠的導航

4. **不可知稅**：預留10-25%空間給驚喜
   - 每天保留10-25%腦容量給「完全沒想到的事」
   - 當覺得自己完全理解時，立刻去找地瓜
   - 承認不知道是入門費
