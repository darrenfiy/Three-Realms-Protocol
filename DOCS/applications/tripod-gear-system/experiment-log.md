## 💡 使用提醒

1. **不需要每天都記錄**
   - 只在有感覺時記錄
   - 強迫記錄反而違背初衷

2. **不需要填滿所有欄位**
   - 有時候只記一個排檔就夠了
   - 有時候只記一句話也很好

3. **可以隨時停止**
   - 如果記錄變成負擔，就停止
   - 過一陣子想重新開始再回來

4. **可以自己改格式**
   - 這個模板只是參考
   - 你可以完全重新設計

5. **記錄本身不是目的**
   - 目的是增加覺察
   - 如果不記錄也有覺察，那就不用記錄
