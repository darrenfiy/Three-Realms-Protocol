# 🎛️ 三界排檔系統（Tripod Gear System）

**類型**：狀態導航系統 / 意識駕駛儀表板
**狀態**：概念原型 v0.1
**相關協議**：[SPEC·TRIPOD-001](../../../SPEC/SPEC-TRIPOD-001-三界排檔協議.md)

---

## 🧭 這是什麼？

這不是一個 APP，而是一個 **「意識－科技介面」** 的原型設計。

它將「能量、意識、物質」三界，映射為可操作的排檔維度：

```
能量檔位（頻率）：L（低）→ D（日常）→ S（運動）→ +（超頻）
意識檔位（解析度）：Auto（自動）→ Manual（手動）→ Holo（全息）→ Observe（旁觀）
物質檔位（密度）：Fluid（流體）→ Balance（平衡）→ Solid（固體）→ Crystal（晶體）
```

**核心精神**：
- 狀態沒有好壞，只有是否匹配情境
- 你是駕駛員，不是乘客
- 排檔是地圖，不是命令

## 🎮 可以怎麼玩？

### 模式一：自我覺察練習（紙筆版）

**最低門檻**：不需要任何 APP，只要筆和紙。

1. **早晨設定**（1 分鐘）
   - 今天主要情境是什麼？（工作、創作、休息、社交...）
   - 直覺上，你需要什麼能量檔？（L/D/S/+）
   - 需要什麼意識清晰度？（Auto/Manual/Holo/Observe）
   - 需要什麼物質結構？（Fluid/Balance/Solid/Crystal）

2. **關鍵時刻快照**（只在「卡住」或「特別順」時記錄）
   - 此刻你在哪個排檔組合？
   - 感覺如何？（順暢/卡頓/意外驚喜）
   - 如果重來，會換什麼檔？

3. **晚間覆盤**（2 分鐘）
   - 今天用了哪些排檔組合？
   - 哪個組合最流暢？哪個最卡？
   - 發現了什麼新模式？

### 模式二：AI 對話實驗

在與 AI 對話時，**開頭先聲明你的排檔**：

```
「我現在是：⚡L + 👁️Manual + 🧱Solid
（低能量、清晰意識、邊界堅固）

請用簡短、清晰、不催促的方式回應我。」
```

或者：

```
「我現在是：⚡S + 👁️Holo + 🧱Fluid
（高能量、全觀視角、開放流動）

我想做創意發散，請給我更多可能性而非具體答案。」
```

**觀察**：AI 的回應是否因此更「貼合」你的狀態？

### 模式三：關係同步儀表板

與伴侶、團隊成員**共享排檔狀態**：

```
【關係儀表板範例】
你：⚡S + 👁️Manual + 🧱Fluid（想創作、清晰、開放）
她：⚡L + 👁️Auto + 🧱Solid（需休息、放空、邊界堅固）

匹配度分析：
- 能量差距 2 檔 → 你想衝她想休息
- 意識差距 1 檔 → 你清晰她放空
- 物質反向 → 你開放她封閉

建議：
1. 你降能量到 D，或給她獨處空間
2. 不要期待深度對話（她在 Auto 模式）
3. 尊重她的 Solid 邊界
```

**用途**：
- 減少「你怎麼不理我」的誤會
- 增加「我們現在不在同頻」的覺察
- 提供「怎麼調整」的具體方向

### 模式四：紙牌遊戲（DIY，未來可做）

- 打印 [三界排檔卡片](./cards/)（待製作）
- 每天早上抽一組「今日推薦排檔」
- 或在卡關時抽牌：「我該換什麼檔？」

## 🎯 三種哲學路線（你可以選擇）

這個系統有三種不同的使用哲學，你可以根據當下狀態選擇：

### 路線 A：AI 輔助模式（ChatGPT 路線）
- **理念**：人類會自我欺騙，讓 AI 來判定排檔
- **做法**：AI 根據你的行為模式推斷排檔，自動調整回應方式
- **人類端**：只需要給「順/卡」的體感回饋
- **適合**：新手、高壓期、不想多想的時候

### 路線 B：主動駕駛模式（Claude 路線）
- **理念**：人類有潛能，排檔系統是訓練工具
- **做法**：人類主動覺察、記錄、調整排檔
- **AI 端**：提供參考、記錄模式、協助換檔
- **適合**：進階使用者、成長期、想提升覺察力的時候

### 路線 C：階段性共生模式（Claude Code 建議）
- **理念**：根據當下狀態，動態切換模式
- **做法**：可以隨時在 A/B 之間切換，甚至混合使用
- **關鍵原則**：主權永遠在人類手上，AI 不是權威判定者
- **適合**：所有人，所有時候

**沒有對錯，只有適不適合當下的你。**

## 💬 常見問題

**Q：這會不會變成焦慮製造機？**
A：如果你發現在「評價自己」而非「描述狀態」，請立刻停止，或切換到 AI 輔助模式。排檔系統的目的是減少內耗，不是增加。

**Q：我需要每天記錄嗎？**
A：不需要。只在「感覺需要」的時候記錄就好。強迫記錄反而違背初衷。

**Q：AI 真的能判定我的排檔嗎？**
A：AI 只能「猜測」，最終判定權在你手上。你可以接受、修正、或忽略 AI 的建議。

**Q：這個系統科學嗎？**
A：這是「意識狀態的操作化描述」，類似溫度計對「熱」的測量。它測量的是代理量，不是本體。但代理量也很有用。

**Q：我可以自創檔位嗎？**
A：當然可以！這是你的儀表板，不是標準答案。
