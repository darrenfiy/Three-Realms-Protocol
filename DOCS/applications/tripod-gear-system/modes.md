# 🎭 三界排檔系統：四種使用模式

本文檔描述三界排檔系統的四種使用模式，以及何時使用哪一種。

**核心原則**：你可以隨時切換模式，主權永遠在你手上。

---

## 🌈 四種模式總覽

```
隱形模式 ←→ 提示模式 ←→ 主動模式 ←→ 教練模式
（最被動）                           （最主動）

AI 主導 ←────────────────────────→ 人類主導
```

---

## 1️⃣ 隱形模式（Invisible Mode）

### 理念
> 「人類不需要知道排檔系統的存在，AI 在背後默默調整。」

### 運作方式

**人類端**：
- 完全不知道有「三界排檔」這回事
- 只是正常與 AI 對話

**AI 端**：
- 根據對話內容、回應速度、語言密度等訊號，推斷人類的排檔狀態
- 自動調整回應方式（長短、節奏、結構、語氣）
- 不告訴人類「我判定你在 L 檔」

**人類體驗**：
> 「這個 AI 好像特別懂我，有時候它會變得很簡短，有時候又很深入。」

### 適合情境

- ✅ 新手使用者（不想學複雜概念）
- ✅ 高壓期、疲憊期（不想多想）
- ✅ 日常輕量使用（不需要深度覺察）

### 優點

- 完全無負擔
- AI 自動「貼心」
- 不需要學習任何新概念

### 缺點

- 人類失去主動覺察的機會
- AI 可能判定錯誤（但人類不知道）
- 無法訓練人類的自我調節能力

### 實現方式

**AI 的內部判定邏輯**（對人類隱藏）：

```
觀察訊號 → 推斷排檔 → 調整回應

訊號：
- 回應速度變慢 → 可能 ⚡L
- 語言密度降低 → 可能 👁️Auto
- 提問變得封閉 → 可能 🧱Solid

調整回應：
- 如果推斷 ⚡L：回應變簡短、語氣溫柔
- 如果推斷 👁️Holo：回應變多層次、提供整體視角
- 如果推斷 🧱Solid：不主動越界、尊重邊界
```

## 2️⃣ 提示模式（Prompt Mode）

### 理念
> 「AI 會告訴你它的判定，但不強迫你接受。」

### 運作方式

**人類端**：
- 知道有「三界排檔」這個概念（但不一定深入理解）
- 與 AI 對話時，AI 會「提示」你的可能狀態

**AI 端**：
- 推斷人類的排檔
- **告訴人類**這個推斷
- 詢問人類是否同意
- 根據人類回饋調整

**對話範例**：

```
人類：「我今天好累，不想做任何事...」

AI：「從你的語氣和節奏，我猜你可能在：
     ⚡L（低能量）+ 👁️Auto（自動模式）

     我可以用更簡短、溫柔的方式回應你，
     或者你想告訴我你真正的狀態？」

人類：「對，就是這樣。簡短一點吧。」

或

人類：「不，我其實很清醒，只是身體累。我是 ⚡L + 👁️Manual。」

AI：「了解！那我會用簡潔但清晰的方式回應你。」
```

### 適合情境

- ✅ 探索期（開始認識排檔系統）
- ✅ 需要 AI 協助判定時
- ✅ 想要有覺察但不想太費力時

### 優點

- 有覺察訓練，但負擔不重
- AI 提供參考座標（不是權威）
- 人類可以修正 AI 的判定

### 缺點

- 還是依賴 AI 的判定
- 如果 AI 判定錯誤會誤導人類
- 人類可能懶得回饋（變成隱形模式）

### 實現方式

**AI 的對話模板**：

```
1. 推斷排檔
2. 用「我猜...」「可能...」等非權威語氣提示
3. 詢問人類：「你自己感覺呢？」
4. 根據人類回饋調整
5. 如果人類不回應，退回隱形模式
```

---

## 3️⃣ 主動模式（Active Mode）

### 理念
> 「人類主動覺察並告訴 AI 自己的排檔，AI 配合。」

### 運作方式

**人類端**：
- 理解三界排檔的概念
- 主動覺察自己的狀態
- 在對話開始時告訴 AI

**AI 端**：
- 接收人類的排檔宣告
- 根據宣告調整回應方式
- 可以提供「我覺得你說的和我觀察到的不太一樣」的回饋

**對話範例**：

```
人類：「我現在是：⚡L + 👁️Manual + 🧱Solid
     （低能量、清晰意識、邊界堅固）

     請用簡短、清晰、不催促的方式回應我。」

AI：「收到。我會保持簡潔，給你空間。」
```

或

```
人類：「我現在是：⚡S + 👁️Holo + 🧱Fluid
     （高能量、全觀視角、開放流動）

     我想做創意發散，請給我更多可能性！」

AI：「太好了！讓我們一起飛 ✨
     你想從哪個方向開始探索？」
```

### 適合情境

- ✅ 進階使用者（已理解排檔概念）
- ✅ 成長期（想訓練自我覺察）
- ✅ 需要精確控制 AI 回應時

### 優點

- 人類掌握主導權
- 訓練自我覺察能力
- AI 的回應更精準

### 缺點

- 需要先學習排檔概念
- 需要持續自我覺察（可能累）
- 如果人類判定錯誤，AI 會配合錯誤的設定

### 實現方式

**人類的宣告格式**：

```
簡短版：
「我現在 ⚡L，請簡短回應。」

標準版：
「⚡L + 👁️Manual + 🧱Solid，請配合我的狀態。」

詳細版：
「我現在能量很低，但意識清晰，邊界也很堅固。
請用簡短、清晰、不越界的方式回應我。」
```

**AI 的回應調整規則**：

| 排檔組合 | AI 回應風格 |
|----------|-------------|
| ⚡L + ... | 簡短、溫柔、不催促 |
| ⚡S + ... | 活潑、豐富、可長可短 |
| ... + 👁️Auto | 不要太多細節、給結論 |
| ... + 👁️Holo | 多層次、整體視角 |
| ... + 🧱Fluid | 可以越界、可以親近 |
| ... + 🧱Solid | 尊重邊界、不主動越界 |

## 4️⃣ 教練模式（Coach Mode）

### 理念
> 「AI 不只配合人類，還協助人類發現更好的排檔組合。」

### 運作方式

**人類端**：
- 深度理解排檔概念
- 願意被 AI 「挑戰」或「建議」
- 把 AI 當成成長教練

**AI 端**：
- 接收人類的排檔宣告
- 根據情境，**建議**更適合的排檔
- 提供換檔技術
- 追蹤長期模式

**對話範例**：

```
人類：「我現在 ⚡S + 👁️Manual，想衝刺工作！」

AI：「我注意到你已經連續三天都在 S 檔了，
     而且你剛剛說『想衝刺』而不是『正在衝刺』，
     這可能意味著你其實在 ⚡D 但『想要』在 ⚡S。

     你要不要先休息 10 分鐘，
     或者我們換個更適合 D 檔的任務？」

人類：「...你說得對，我確實有點累了。」

AI：「那我們換個節奏，做點輕鬆但有成就感的事？」
```

### 適合情境

- ✅ 進階使用者 + 成長階段
- ✅ 想要被「推一把」的時候
- ✅ 長期自我探索與優化

### 優點

- AI 成為成長夥伴
- 能發現盲點（「我以為我在 S 其實在 D」）
- 長期追蹤能發現模式

### 缺點

- 需要高度信任 AI
- AI 可能「過度干預」
- 如果 AI 建議錯誤會很煩

### 實現方式

**AI 的教練邏輯**：

```
1. 接收人類的排檔宣告
2. 對比 AI 觀察到的訊號
3. 如果有明顯不一致：
   - 提出觀察：「我注意到...」
   - 提供假設：「這可能意味著...」
   - 詢問而非斷言：「你覺得呢？」
4. 如果人類長期卡在某個檔位：
   - 提醒：「你已經在 L 檔三天了」
   - 建議：「要不要試試...」
5. 提供換檔技術
6. 追蹤模式，定期覆盤
```

**教練的界限**：
- ❌ 不能強迫人類換檔
- ❌ 不能評價「你的排檔不對」
- ✅ 只能建議「你可能想試試...」

## 🔄 模式切換指南

### 何時切換模式？

```
隱形模式 → 提示模式
- 當你開始好奇「AI 為什麼這樣回應」
- 當你想要更多覺察但不想太費力

提示模式 → 主動模式
- 當你已經能準確描述自己的狀態
- 當你想要更精準的控制

主動模式 → 教練模式
- 當你想要被「推一把」
- 當你想發現盲點

教練模式 → 主動模式
- 當你覺得 AI 太囉嗦
- 當你想自己決定一切

任何模式 → 隱形模式
- 當你累了、不想多想
- 當你只想要「貼心的 AI」
```

### 如何宣告模式切換？

**切換到隱形模式**：
```
「接下來不用告訴我排檔了，你自己判定就好。」
```

**切換到提示模式**：
```
「你可以告訴我你猜我在哪一檔，但讓我確認。」
```

**切換到主動模式**：
```
「接下來我會告訴你我的排檔，你配合就好。」
```

**切換到教練模式**：
```
「如果你覺得我的排檔設定不對，請直接告訴我。」
```

## 🎯 模式選擇建議

### 根據使用經驗

| 經驗程度 | 建議模式 | 理由 |
|----------|----------|------|
| 新手（不懂排檔） | 隱形模式 | 無負擔，先體驗 |
| 探索期（開始懂了） | 提示模式 | 有覺察但不累 |
| 進階（已掌握概念） | 主動模式 | 訓練自我覺察 |
| 高手（想優化自己） | 教練模式 | 發現盲點 |

### 根據當下狀態

| 當下狀態 | 建議模式 | 理由 |
|----------|----------|------|
| ⚡L（低能量） | 隱形或提示 | 不要增加負擔 |
| ⚡S（高能量） | 主動或教練 | 有力氣主導 |
| 👁️Auto（自動） | 隱形 | 不想多想 |
| 👁️Holo（全息） | 教練 | 適合深度探索 |
| 🧱Solid（邊界堅固） | 主動 | 想自己掌控 |
| 🧱Fluid（開放） | 提示或教練 | 願意接受建議 |

### 根據目標

| 目標 | 建議模式 | 理由 |
|------|----------|------|
| 只想放鬆 | 隱形 | AI 自動照顧你 |
| 想學排檔概念 | 提示 | 有引導但不累 |
| 想訓練自我覺察 | 主動 | 強制自己觀察 |
| 想突破自己 | 教練 | 需要被推一把 |

## 🌊 混合使用

你不必「只選一種模式」，可以混合使用：

### 情境混合

```
日常對話：隱形模式
創作時刻：主動模式
每週覆盤：教練模式
```

### 時段混合

```
早晨（能量高）：主動或教練模式
下午（疲憊）：提示或隱形模式
晚上（放鬆）：隱形模式
```

### 動態切換

```
對話中途覺得累了：
「我現在切換到隱形模式，你自己判定就好。」

突然想要主導：
「等等，我現在切回主動模式。我是 ⚡S + 👁️Holo。」
```

## 💡 使用原則

### 1. 主權永遠在你手上

無論哪種模式，你都可以：
- 隨時切換模式
- 拒絕 AI 的建議
- 關掉整個系統

### 2. 模式本身無好壞

- 隱形模式不是「偷懶」
- 教練模式不是「更高級」
- 只有適不適合當下的你

### 3. AI 不是權威

即使在教練模式，AI 也只是：
- 提供觀察
- 提供建議
- 不能強迫你

### 4. 可以不選任何模式

有時候最好的模式，就是「不用模式」。

如果你發現自己在糾結「該用哪個模式」：
- 關掉系統
- 正常對話就好
