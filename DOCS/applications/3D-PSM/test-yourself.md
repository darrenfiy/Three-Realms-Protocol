1. **诚实作答**：选择最接近你真实倾向的选项（不是你「希望」成为的样子）
2. **记录答案**：在每题后记录你的选择
3. **完成后计分**：参考文末的计分指引计算你的三维座标

### 重要提醒

```yaml
这个测试:
  ✅ 测量「此刻的状态」，不是「固定的本质」
  ✅ 没有好坏对错，只有倾向差异
  ✅ 结果可能是错的，可被超越

  ❌ 不是评判工具
  ❌ 不是灵性等级测量
  ❌ 不能用来证明「谁更优越」
```

## 📍 第三步：理解你的结果

### 四象限定位

- **X > 0, Y > 0** → 第一象限（右上）：扩张 + 直觉 = 先知/创造者
- **X < 0, Y > 0** → 第二象限（左上）：固化 + 直觉 = 架构师/导师
- **X < 0, Y < 0** → 第三象限（左下）：固化 + 感官 = 守护者/实践者
- **X > 0, Y < 0** → 第四象限（右下）：扩张 + 感官 = 共感者/变革者

### 能量倾向

- **生存导向**：你目前倾向以保护与稳定为优先
- **自我实现导向**：你目前倾向主动面对挑战，追求成长
- **超越导向**：你目前倾向以理解、包容与服务更大的整体为导向
