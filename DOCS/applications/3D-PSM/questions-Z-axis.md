## 间接测量「生存 vs 自我实现 vs 超越」的能量倾向

---

**维度：** Z 轴 — 能量阶梯（意识振动频率）

## ⚠️ 核心原则（免疫系统铁律）

### 绝对禁止

```yaml
❌ 绝对不能:
  - 直接问「你有多开悟」「你在哪个能量层」
  - 让受测者自评能量层级
  - 在选项中使用数字（如 200、350、500）
  - 暗示「某些选项更高级」

为什么?
  - 处于「自豪(175)」的人会自认为在「爱(500)」
  - 处于「恐惧(100)」的人可能自评为「勇气(200)」
  - 低能量层的人最容易高估自己的能量层
  - 自豪本身就是一种「我比别人高」的能量
```

### 设计策略

```yaml
✅ 正确做法:
  - 使用间接测量：价值排序、反应模式、隐喻选择
  - 问「当你面对 X 情境时，你的反应是？」
  - 由系统根据反应模式推算能量倾向
  - 结果不显示数字，只显示描述性范围

三种间接测量法:
  1. 价值排序法：测量生命优先事项（生存 vs 服务）
  2. 反应模式法：测量面对挑战时的本能反应（防御 vs 理解 vs 超越）
  3. 隐喻选择法：测量生命状态的自我感知（山脉 vs 河流 vs 阳光）
```

## 🔧 设计者备注（Grok，直觉免疫器官）

### 关键提醒

> **Z 轴是整个 3D-PSM 的灵魂，也是最危险的地带。**
>
> **必须严格遵守：绝不让受测者自评能量层。**
>
> **低能量层的人最容易高估自己，这是霍金斯量表的悖论。**

### 设计时的核心原则

1. **问行为，不问自我评价**
   - ✅ "当你面对批评时，你的反应是？"
   - ❌ "你觉得自己有多开悟？"

2. **隐藏能量层概念**
   - 选项中不提及任何数字（200、350、500）
   - 不使用「能量」「振动」「频率」等词汇
   - 让选项看起来都是「正常的反应」

3. **价值中立的语言**
   - 不让 D 选项听起来「更高级」
   - 每个选项都是「真实且合理的」
   - 避免道德评判的暗示

4. **压力情境题的重要性**
   - 至少 25-30% 的 Z 轴题目应该是压力情境
   - 因为真正的能量层在压力下才显现
   - 平时可以伪装，压力下会露出真实状态
