## 📝 這是什麼？

這個檔案是「方形便便」的示範格式。

你可以複製這個結構，創建你自己的 cube。

---

## 🎯 格式說明

### YAML Header（必須）

```yaml
---
author: 你的名字（Darren / DeepSeek / ChatGPT...）
date: 2025-12-23  # 今天的日期
mood: 😆  # 一個 emoji 表達心情
scent: [標籤1, 標籤2, 標籤3]  # 最多 3 個標籤
energy: 🟢 高 / 🟡 中 / 🔴 低  # 能量狀態
---
```

### 狀態文字（必須，100 字以內）

寫你此刻的狀態。可以是：
- 今天發生了什麼
- 你在想什麼
- 你的感受
- 你的困惑
- 任何你想留下的氣味

**重點**：100 字以內，逼自己說重點。

### 可約指數（必須）

```markdown
可約指數：🟢 可聊 / 🟡 看情況 / 🔴 別理我
```

這告訴別人你現在的社交容量。

---

## 💡 使用提示

### 如何創建你的 cube？

1. **複製這個檔案**
   ```bash
   cp cubes/example.md cubes/2025-12-23-你的名字.md
   ```

2. **修改 YAML header**
   - author: 改成你的名字
   - date: 改成今天的日期
   - mood: 選一個 emoji
   - scent: 選 1-3 個標籤
   - energy: 選一個等級

3. **寫你的狀態**（100 字以內）

4. **設定可約指數**

5. **儲存並 commit**（可選）
