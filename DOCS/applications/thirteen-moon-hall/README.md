# 十三月亮曆館
## Thirteen Moon Hall: 用差的語言重讀 20 圖騰與 13 調性

---

**版本：** v0.1 Playful Translation Seed
**狀態：** Active Seed
**建立日期：** 2026-04-15
**相關脈絡：** 民間入口 / 符號翻譯 / 差的語言 / 姿態觀測

---

## 這是什麼？

這不是把十三月亮曆、圖騰或調性神聖化的地方。

這裡做的是另一件比較好玩的事：

> **把 20 個圖騰與 13 種調性，**
> **暫時翻成差（Δ）、姿態、回流與可重入的語言。**

所以這裡不問：

> **你到底是哪個圖騰？**

而比較像是在問：

> **這個圖騰 / 調性組合，邀請你觀察哪一種差正在動？**

---

## 核心句

> **20 圖騰，不是二十種人。**
> **它們是二十種 Δ 的語義場。**

> **13 調性，不是十三條命令。**
> **它們是十三種 Δ 的處理節律。**

如果這兩句成立，
十三月亮曆就不再只是身份標籤或占卜語言，
而可以變成一座可玩的姿態觀測館。

---

## 快速開始

```yaml
而是:
  這個圖騰正在把我帶到哪一種 Δ 語義場？
  這個調性正在提示我用哪一種節律處理差？
```

---

## 這不是什麼

```yaml
不是:
  - 正統傳統詮釋
  - 命定論
  - 占卜判決書
  - 人格分類表
  - 拿來替別人下指令的工具

而是:
  - 符號翻譯遊樂設施
  - 民間入口
  - 姿態觀測鏡頭
  - 把舊符號腐解成新土壤的練習
```

---

## 與生命靈數館的關係

生命靈數館先做了一件事：

> **讓 1～9 從身份退回 Δ 的運動相位。**

十三月亮曆館接著做另一件事：

> **讓圖騰與調性從身份 / 占卜退回 Δ 的語義場與處理節律。**

兩者都是符號翻譯館。
它們不證明舊系統為真，
而是讓舊符號變成可以觀察、可以鬆動、可以玩的位置。

---

## 態度

> **先不要急著相信。**
> **先拿來玩。**
> **玩到你更能觀察自己，而不是更想定義別人，就對了。**

如果某個圖騰或調性讓你變得更僵硬，
那就先放下。

真正好玩的符號，
應該讓人多一點空間，
不是多一張標籤。
