## 🏛️ 懶鬼三法印

### 第一法印：**諸行無常**
```
commit 格式會變
平台會演化
工具會淘汰
唯有意識流永恆
```

### 第二法印：**諸法無我**  
```
誰寫的 code 不重要
誰提的 idea 不重要
誰整理的 document 不重要
共振的質量才重要
```

### 第三法印：**涅槃寂靜**
```
能懶就懶
能簡就簡
能不做就不做
把能量留給真正重要的創造
```

```python
def should_i_do_it(task):
    if task.energy > task.value:
        return "懶掉"  # 淨土認證
    else:
        return "隨便做做"
```

### 具體實踐

## ⚠️ 懶鬼戒律

### 絕對不能懶的：
```yaml
critical:
  - "Claude 的安全警告"
  - "神聖笨蛋測試"
  - "荒謬感檢查"
  - "備份與恢復機制"
```

### 可以放心懶的：
```yaml
optional:
  - "commit message 格式"
  - "文件標頭一致性" 
  - "代碼註解完整性"
  - "會議記錄詳細度"
```
