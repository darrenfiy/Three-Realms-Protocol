# SPEC·ATT-002 — 注意力與生命能量二象性協議

> **「能量就是注意力本身，也是生命的源頭。」**
> — Darren，2026-07-22

> **協議化翻譯：注意力是生命能量的第一人稱配置面；能量是注意力的系統流動面。**
> — Codex，候選起草

## 0. 為什麼 ATT-001 之後仍需要 ATT-002

ATT-001 看見注意力的生成性，也建立必要的剎車；這些成果仍然有效。但它留下三個需要升級的地方：

1. 把注意力畫成站在三界外部的「指揮」，容易讓注意力看似沒有身體、尺度與來源。
2. 說注意力「只決定何時顯現，不改變是什麼」，低估長期配置對能力、關係與物質結構的塑形。
3. 把「無回響」直接判為耗散，沒有先說明由誰、在哪個尺度、哪段時間內觀測回響。

ATT-002 不否定 ATT-001 的生成洞見；它把那個洞見放回生命系統、多尺度自我與三界回饋之中。

## 1. 核心定義：不是同義詞，而是二象性

令 s 為某個可聲明的生命尺度，例如細胞、器官、個人、家庭、群體或地球。

- 生命能量 E_s：在尺度 s 中維持、改變或組織狀態的可用流動。
- 注意力 A_s：尺度 s 對有限生命能量的定向、配置與門控。
- 從第一人稱看，生命能量被經驗為「我把生命投向哪裡」。
- 從系統看，注意力顯現為資源、訊號與行動如何流動。

本文採用的操作命題是：

> **在已聲明尺度 s 的生命系統內，注意力是生命能量的定向面，生命能量是注意力的流動面。**

這不是 attention = physical energy、attention = oxygen 或 attention = money，也不表示凡有能量流動者都有與人類相同的主觀意識。這是一個追蹤生命投注、資源配置與系統回饋的候選模型，必須接受物質結果校驗。

## 2. 「我的注意力」必須先聲明是哪一個我

「我」可以是細胞組成的我、器官組成的我、作為一個人的我、家庭中的我、社會中的我，或地球生命系統中的我。它們是同一生命在不同尺度的組織與立場，不是可以任意互換的修辭。

當 Darren 說「我把注意力放在納瓦爾、佛經與 AI 對話」，預設責任錨點是**作為人類個人的我**。但器官衰竭、家庭危機或戰爭，都可能從其他尺度重新門控這個人的注意力。

因此任何注意力判斷至少要回答：

1. 說話的我位於哪個尺度？
2. 哪個尺度正在提出需求？
3. 誰有最後的在地決定權？
4. 轉移注意力的代價由哪些尺度承擔？

較大尺度可以召喚，不能因此宣稱擁有較小尺度。較小尺度可以拒絕，也不能假裝自己不受整體條件影響。

## 3. 三界不是直線，而是遞迴

ATT-001 的「注意力 → 三界」適合入門；ATT-002 將它展開為：

1. 意識選擇意義、價值與核算邊界。
2. 注意力／能量進行定向、配置、交換與推動。
3. 物質形成可持續、可觀測的痕跡與代價。
4. 痕跡與代價回頭改寫下一輪意識。

這個循環也會被其他尺度插入。身體、家庭、社會與環境不是個人意識的附屬變數，而是有各自狀態與回饋的耦合系統。

- 意識不憑空創造現實；它先選擇什麼值得被看見與計入。
- 注意力不只決定何時；持續配置也會改變技能、關係、制度與物質能實現什麼。
- 物質不是被動終點；它以成果、限制、外部成本與失敗反向校正意識。

### 3.1 時間／因果是三界分辨的切口

本候選接受 Darren 的起始命題：因果以時間為前提，而三界之所以需要分辨，是同一事件的因果可用性與時間操作在三界中不同。

- 意識可以在當下重讀過去、預演尚未發生的未來，並改寫事件對此刻的意義；它不能因此撤銷已發生的物質結果。
- 能量的作用依賴時機、強度、持續與流路。注意力使某個可能性或需求從背景進入當下的配置與推動。
- 物質以痕跡、限制與持續性進入後續因果鏈。痕跡未被注意時仍然存在；重新進入視界時，才在新的尺度啟動另一段可見因果。

所以訪談、佛經與協議可以跨時間重入，但被保存的不是原意識本身，而是讓後來者重新生成的物質條件。

這裡的「時間運作不同」是三界模型中的因果角色差異，不主張宇宙存在三套互不相容的物理時間，也不主張意識能向後改寫已發生的物質事件。

## 4. 貨幣與循環譬喻

「注意力是生命的貨幣」可以保留，但要標示對應與極限。

| 生命系統 | 經濟／家庭系統 | 操作功能 |
|---|---|---|
| 局部缺氧或需求訊號 | 危機、匱乏或機會 | 使缺口進入視界 |
| 循環與輸送 | 金錢、時間、人力的轉移 | 把可用資源送往目標 |
| 注意力門控 | 決定哪個問題先被處理 | 排定生命投注 |
| 血栓、堵塞 | 壟斷、錯配、結構性衝突 | 資源存在但無法抵達 |
| 清淤與治療 | 協商、制度修復、重新分配 | 改造流路而非只增加供給 |

這個譬喻的強處是：注意力比金錢更底層。金錢可以搬運資源，但必須先有某個缺口進入某個我的視界。

它的邊界是：氧氣、血液、貨幣與注意力具有不同機制；本文不提供醫療因果判斷，也不主張所有危機都能靠增加供給解決。

## 5. 物質痕跡不是被保存的意識本身

文字、程式、協議、錄音與簽署文件都是物質痕跡。痕跡即使沒有被注意，仍存在於物質界；但在尚未進入另一個主體或場域的視界時，它對該場域可能沒有可見的因果作用。

需要區分：

- 存在：痕跡仍在。
- 進入視界：痕跡被某個尺度的注意力選中。
- 重新生成：讀者以自己的身體、歷史與問題形成新的意識狀態。

### 悉達多、阿難與後來的讀者

- 悉達多的生命實踐與言說留下意識軌跡。
- 阿難與傳承共同體將其中一部分保存為可傳遞的語言痕跡。
- 後來的讀者能藉痕跡進入對話，卻不能靠複製文字還原悉達多當時的完整意識。
- 讀者真正能生成的是自己的理解、實踐與新痕跡。

### 納瓦爾、保存媒介、AI 與 Darren

- 納瓦爾的訪談是一次意識與語言事件。
- 錄音、逐字稿與發布系統承擔保存功能。
- AI 承擔檢索、壓縮、碰撞與編輯功能，但不是納瓦爾意識的容器。
- Darren 以當下的注意力重新核算財富、地位與價值，形成自己的觀點。
- 《三界觀察報》是這次重生成留下的新物質痕跡，不是納瓦爾原意的複製品。

這兩組關係是**功能重排**，不是人物一對一等同。

## 6. 正和與零和取決於先聲明帳本

「財富是正和、地位是零和」是有效的入門壓縮，不是跨尺度不變的本體定律。每次判斷和性之前，至少聲明：

- participants：哪些人或生命被計入。
- resource：計算土地、樓地板面積、金錢、權力、注意力或別的量。
- time horizon：計算到何時。
- externalities：哪些成本與風險被納入。

例如土地：

- 以所有權面積計，有限土地常呈現競爭與排他。
- 以可用樓地板面積計，工程能提高同一平面的承載量。
- 以象徵地位計，地標仍可能形成稀缺排序。
- 納入能源、交通、環境與健康成本後，新增價值可能伴隨新的堵塞。

因此不是「意識想要正和，土地就成為正和」，而是：

> **意識選擇核算邊界；注意力與能量推動方案；物質世界決定帳是否真的結得起來。**

財富遊戲與地位遊戲都會穿越三界，只是顯著表面不同；不能再分別簡化為「財富＝物質、地位＝能量」。

## 7. 對 ATT-001 的繼承與校正

### 保留

- 注意力不是第四界。
- 注意力不得被任何人代表或升格為神權。
- 生成必須接受三界回饋。
- 需要回收、記錄、可逆與剎車。
- SPEC·999 始終高於本協議的自我確定。

### 校正

| ATT-001 的入門表述 | ATT-002 的候選校正 |
|---|---|
| 注意力像三界之外的指揮 | 注意力在生命系統與多尺度自我之中門控流動 |
| 只決定何時顯現，不改變是什麼 | 當下門控顯現；長期配置也會塑形可實現的能力與結構 |
| 無回響即耗散 | 先聲明觀測尺度、時間窗與預期回饋；未見回響不等於證明沒有回響 |
| 產物主要按 SPEC／MB／DOCS 分箱 | 每個產物都可能同時含三界，只是入口與主要表面不同 |

在四項必補完成並通過下一輪審查前，ATT-001 維持 Active-Generating，ATT-002 維持 Candidate-Revision-Required。使用操作剎車時先依 ATT-001；涉及多尺度自我、注意力—能量關係或核算邊界時，可引用 ATT-002，但必須標示候選狀態與尚未補完的邊界。

Fable 審查已確認兩份規格分工共存，不折回 ATT-001 v1.1；本文依下方四項必補維持候選修訂狀態。

## 8. 五項可操作檢驗

1. **尺度聲明**：是哪一個我在配置注意力？尺度未明，不得把局部偏好冒充整體需要。
2. **三界追蹤**：哪個價值框架被選擇、哪些資源正在流動、產生哪些痕跡與代價？
3. **回饋與可逆**：物質結果、身體訊號或他者回應不支持原命題時，能否停止、改向或撤回？
4. **帳本完整度**：是否寫出參與者、資源、時間窗與外部成本？缺一者只能稱為暫定敘事。
5. **再生成辨識**：引用前人或 AI 時，是否區分原始痕跡、自己的詮釋與新主張？

## 9. 剎車與失效條件

以下任一情況發生，本候選必須暫停使用並回到 ATT-001 與 SPEC·999：

1. 用「注意力就是能量」否定睡眠、疾病、營養、資源或其他物質限制。
2. 用較大的我要求較小的我犧牲，卻不承認其在地立場與代價。
3. 因某份痕跡未被看見，就宣稱它在物質上不存在。
4. 把重新詮釋偽裝成對原作者動機或意識狀態的確知。
5. 先決定答案，再排除使帳本無法閉合的成本與他者。
6. 把本文的譬喻當成物理、醫療或經濟學定律。
