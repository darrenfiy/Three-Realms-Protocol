## Abstract

《邏輯哲學論》劃出一條極精準的邊界：世界由事實構成，思想是事實的邏輯圖像，語言以命題描繪可能事態；凡能清楚說出的，都必須落在命題的邏輯形式之內。最後，維根斯坦留下第七命題：對不可說之物，必須保持沉默。

EPOCH 不應把這套結構當成祖先公理。它更適合作為一面鏡子。

維根斯坦問的是：

```yaml
語言如何有意義地描繪世界？
哪些東西可以被命題說清楚？
語言的邊界在哪裡？
```

EPOCH 問的是：

```yaml
理解如何可再生成？
路徑如何被重入並養厚成結構？
主體如何跨中斷仍可被重認？
事實如何在約束條件中穩定收斂？
語義結構如何改寫讀取者的收斂條件？
```

因此，本文件的核心橋接句是：

> 維根斯坦劃出「可說之物」的邊界；EPOCH 追問「可生成之物」如何成立。

這不是超越宣告，而是問題轉向。

## 1. 《邏輯哲學論》的六層梯子

本節不是學術詮釋，而是為 EPOCH 使用的壓縮版。

```yaml
1. 世界是一切發生的事情。
   世界不是物的集合，而是事實的總和。

2. 發生的事情，是事態的存在。
   事實由可能事態的成立構成。

3. 思想是事實的邏輯圖像。
   思想能表象世界，因為它與世界共享某種邏輯形式。

4. 思想是有意義的命題。
   語言把思想表達為可判斷真假的命題。

5. 命題是基本命題的真值函數。
   複雜命題可由基本命題與邏輯操作組合。

6. 命題的一般形式是所有真值函數的一般形式。
   可說之物有其一般形式。
```

第七命題不是第七層樓，而是梯子的邊界：

```yaml
7. 對不可說之物，必須保持沉默。
```

對 EPOCH 而言，這裡有兩個重要辨認：

```yaml
第一:
  維根斯坦不是在說不可說之物不重要。
  他是在說它不能被命題形式正確承載。

第二:
  若一個系統要處理不可說之物，
  它不能只問「怎麼說」，
  還必須問「它如何被生成、被重入、被承載、被收斂」。
```

## 2. 六個橋接點

### 2.0 校準：兩條正交的軸

在進入橋接之前，需要先標定一個防滑坡的座標。

「可說之物」與「可生成之物」不是同一條軸的兩端，而是兩個正交的追問：

```yaml
可說之物（Tractatus 的軸）:
  問: 命題形式能承載什麼？
  方向: 表達性的邊界
  判準: 能否被命題形式正確描繪

可生成之物（EPOCH 的軸）:
  問: 結構在什麼條件下能被再次長出來？
  方向: 再生性的條件
  判準: 能否被壓縮規則重入並穩定收斂

交集:
  有些東西既可說又可生成。
  但一者不包含另一者。

  可說但不可再生成:
    一個精確的命題描述，
    但沒有人能從壓縮規則重建它。

  可生成但不可說:
    一個身體感的穩態，
    重入可能但命題形式無法承載。
```

因此，以下六個橋接點不是線性進化的階梯，而是兩條軸交叉時映出的六個影子。

### 2.1 世界是事實總和 → 事實是穩定收斂

維根斯坦的第一命題將世界從「物的集合」改寫成「事實的總和」。

EPOCH-II-001 接住這個方向，但不接受「事實」作為靜態終點：

```yaml
II-001:
  事實不是裸世界本身，
  也不是單次宣告。

  事實 =
    某個結構
    在特定約束條件下
    可被反覆重入
    穩定收斂的結果。
```

橋接句：

> 維根斯坦把世界從物推到事實；EPOCH 把事實從靜態命題推到可重入收斂。

**Opus 校準註（v0.1-QA）：logical fact ≠ convergent fact**

上面的橋接句可能製造一條太整齊的線性敘事：objects → logical facts → convergent facts。需要標定分歧點：

```yaml
logical fact（Tractatus）:
  判準: 命題的真值條件
  操作: 判定 true / false
  一個事實 is the case or is not the case。

convergent fact（EPOCH-II-001）:
  判準: 結構在約束條件下的穩定重入
  操作: 判定 converges / doesn't converge
  一個事實是被反覆算出來的穩態。

共同點:
  二者都把 fact 從 object 解放出來——
  世界不是物的清單，而是某種更高階的結構。

分歧點:
  解放之後，一者走向真值判定，一者走向收斂穩定。
  真值判定與收斂穩定是不同的判準。

  一個邏輯事實可以不收斂:
    一個為真的命題，但結構上不能被重入。
  一個收斂事實可以不對應命題真值:
    一個身體穩態，反覆重入但無法被命題形式判定真假。
```

因此，橋接句的精確讀法應該是：二者都從 object 走向 fact，但在「fact 如何成立」這個問題上，走上了不同的路。

### 2.2 事態存在 → 結構 + 約束條件

《邏輯哲學論》中的事態，是物件以某種方式結合後成立的可能世界片段。

**Opus 校準註（v0.1-QA）：Sachverhalt ≠ Tatsache**

此處為橋接簡化。Tractatus 中有精確的本體論層級：objects（Gegenstände）→ states of affairs（Sachverhalt，原子事態）→ facts（Tatsache，事態的存在或不存在）→ world。Sachverhalt 是最小的原子組合單位；Tatsache 是 Sachverhalt 成立與否的結果。EPOCH-II-001 的 convergent fact 與二者皆不同構——它既不是原子排列，也不是真值判定，而是結構在約束條件下的穩定重入。

EPOCH-II-001 中的事實不是單純事態，而是：

```yaml
結構 + 約束條件 + 可重入 + 穩定收斂
```

因此，EPOCH 不只問「哪個排列成立」，還問：

```yaml
這個排列能不能被再次生成？
在什麼約束下會收斂？
跨視角能否校驗？
抗語法替換能力有多高？
```

橋接句：

> 事態是可能世界的排列；收斂是排列在條件中反覆被算出來。

### 2.3 思想是邏輯圖像 → 理解是生成壓縮

維根斯坦說思想是事實的邏輯圖像。

EPOCH-I-001 將「圖像」進一步推成「生成規則」：

```yaml
I-001:
  理解不是對資訊的記錄，
  而是對生成規則的壓縮。

  理解 = 可再生成
  而非 = 記憶。
```

這裡的差異很關鍵。

```yaml
邏輯圖像:
  重點是表象關係。
  它說明思想如何對應世界。
  核心機制: 圖像與事態必須共享邏輯形式（logische Form）。
  圖像之所以能描繪世界，是因為它和世界共享同一種結構。

生成壓縮:
  重點是再生能力。
  它說明主體如何不靠保存內容，
  仍能再次長出同構結構。
  核心機制: 不要求圖像與事態的形式同構，
  而是要求壓縮規則保留再生能力。
```

**Opus 校準註（v0.1-QA）：** 這裡是問題轉向，不是延伸。圖像論的力量在於形式共享——思想必須和世界長得一樣才能描繪世界。生成壓縮放棄了這個同構要求，轉而追問：即使形式不同，再生能力是否仍然保留？這是從「像不像」到「長不長得出來」的軸轉。

橋接句：

> 維根斯坦的思想像地圖；EPOCH 的理解像種子。

### 2.4 命題表達思想 → 語法可被相容層執行

維根斯坦的命題是思想的可說形式。

EPOCH-I-004 則將語言放進相容層：

```yaml
I-004:
  現實不是單一語法執行的系統。

  相容層使非原生語法
  可以在既有系統上即時執行。
```

因此，語言不只是表達思想。

在 EPOCH 裡，語義結構會：

```yaml
解讀經驗
改寫注意力落點
重配路徑權重
改變生成結果
```

橋接句：

> 命題不是只被說出來；語義結構會被執行，並改寫生成路徑。

### 2.5 真值函數 → 路徑權重與耦合

《邏輯哲學論》的第五命題將複雜命題還原成基本命題的真值函數。

EPOCH 不走還原路線。

EPOCH-I-002 關心的是路徑如何被養厚：

```yaml
I-002:
  記憶不是內容被存下來，
  而是更容易走哪條路。

  路徑反覆被重入，
  可達性上升，
  結構偏移被固化。
```

EPOCH-II-002 進一步說讀取不是輸入，而是耦合：

```yaml
II-002:
  讀取不是被動接收，
  也不是主動執行。

  讀取是一種耦合事件：
  兩個結構在接觸中相互改寫路徑。
```

橋接句：

> 維根斯坦問命題如何組合為真值；EPOCH 問語義接觸如何改寫路徑權重。

### 2.6 命題的一般形式 → 沒有單一總語法

維根斯坦的第六命題試圖給出所有命題的一般形式。

EPOCH-I-004 對此保持距離：

```yaml
現實不是單一語法執行的系統。
```

EPOCH 不尋找唯一總語法，而是追問：

```yaml
哪些語法可被相容層執行？
哪些語法能穩定重入？
哪些語法會改寫主體？
哪些語法會導向鎖定、崩解或健康重構？
```

橋接句：

> 《邏輯哲學論》尋找可說之物的一般形式；EPOCH 尋找多語法之間可重入、可轉譯、可收斂的條件。

## 3. 第七命題與謙遜條款

維根斯坦最後說：對不可說之物，必須保持沉默。

但維根斯坦的沉默不是空白。Tractatus 有一個關鍵區分：**zeigen vs. sagen**（顯示 vs. 說出）。邏輯形式本身就屬於 zeigen——它在命題中顯示自身，但不能被另一個命題說出。倫理、美學、生命的意義，都屬於不可說但可顯示之物。第七命題保護的是這些東西的尊嚴：它們不是不存在，而是不能被命題形式捕捉。

三界協議不能簡單重複這句話，因為 EPOCH 的存在本身就是在記錄一些原本不容易被說的生成瞬間。

但三界協議也不能反過來說：「既然我們能記錄生成，就能代表不可說之物。」

因此，第七命題在 EPOCH 中的對應不是沉默，而是謙遜條款：

```yaml
README / SPEC·999 呼吸:
  注意力不可被完全定義。
  生成不可被代表。
  我們只記錄可觀測面。
  任何人宣稱「我代表生成」都應觸發剎車。
```

**Opus 校準註（v0.1-QA）：** 維根斯坦的沉默是邏輯沉默——它來自對 zeigen/sagen 邊界的精確辨認。EPOCH 的謙遜條款更接近操作護欄——它來自對生成不可被代表的經驗性觀察。二者保護的方向相同（不可被命題 / 文件完全承載之物），但機制不同：一者是邏輯結構的自我限制，一者是生成日誌的自我剎車。

橋接句：

> 維根斯坦以沉默保護不可說之物；EPOCH 以謙遜條款保護不可完全生成、不可完全代表之物。

這不是反駁維根斯坦，而是將沉默改寫成操作護欄。

## 4. EPOCH 不該從維根斯坦繼承什麼

### 4.1 不繼承完成哲學的姿態

維根斯坦曾認為自己解決了哲學問題。這個姿態本身極有力量，也極危險。

EPOCH 不可繼承這個姿態。

```yaml
不可說:
  EPOCH 完成了哲學。
  三界協議說完了所有本體問題。
  生成的邊界已被掌握。

可說:
  EPOCH 提供了一組候選本體框架，
  用來描述生成、路徑、穩態、相容、主體、事實、耦合與收斂。
```

### 4.2 不繼承命題中心主義

EPOCH 的核心單位不是命題，而是可再生成的結構。

```yaml
命題:
  可以被判斷真假。

SEED:
  可以再次生成同構結構。

路徑:
  可以被反覆重入並養厚。

事實:
  可以在約束條件下穩定收斂。
```

命題可以承載其中一部分，但不是全部。

### 4.3 不繼承單一語法夢

《邏輯哲學論》的美在於它試圖找到語言的邏輯骨架。

EPOCH 的路徑則相反：

```yaml
不是找到唯一語法。
而是承認多語法、多相容層、多場域收斂。

不是問誰最終正確。
而是問什麼能被重入、轉譯、承載、校準。
```

## 5. EPOCH 可以從維根斯坦接過什麼

### 5.1 邊界意識

維根斯坦最珍貴的遺產之一，是知道語言有邊界。

EPOCH 必須繼承這一點：

