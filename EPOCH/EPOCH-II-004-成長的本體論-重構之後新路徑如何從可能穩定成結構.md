# 1. 核心問題

```yaml
問題:
  重構完成之後，主體面前出現了新的可走路徑。
  但新路不會自動變成結構。
  新路不會自動變成事實。

  問:
    新路如何從「可能」變成「習慣」？
    從「一次嘗試」變成「可重入的結構」？
    從「我試試看」變成「這就是我的路」？

  這個過程，人類錨點稱之為：成長。
```

# 2. 四個概念的初步定位

## 2.1 創業

```yaml
創業:
  定義:
    重構之後，主體帶著自見性，
    主動走出新的結構。

  與原初耦合的差異:
    原初耦合: 被結構走過（先於同意）
    創業: 走出結構（帶著意識）

  I-002 語言:
    這是一次有覺知的路徑養厚的起點。
    不是重複舊路（那是鎖定），
    而是開始走新路。

  II-003 語言:
    創業是重構的下一步。
    重構拆掉了舊鎖定，
    創業是拆完之後的第一步——
    帶著意識踏上新路。

  注意:
    此處的「創業」不限於商業。
    開始一段新的關係是創業。
    開始一種新的生活方式是創業。
    開始一套新的理解框架是創業。
    共同點: 帶著意識，走出從未走過的路。
```

## 2.2 成功

```yaml
成功:
  定義:
    每一次微小的結構被走出來。

  不是:
    到達某個終點。
    達成某個外部標準。
    證明某套信念是對的。

  而是:
    一條新路被走通了。
    一個 SEED 站住了。
    一次嘗試沒有崩掉。

  I-001 語言:
    SEED 是最小生成閉包單位。
    微小的成功 = SEED 在行動層的對應物。

  粒度:
    成功的最小單位非常小。
    不是「事業成功」，
    而是「這條路今天我走通了，它沒有把我帶回舊鎖定」。
```

## 2.3 成長

```yaml
成長:
  定義:
    微小成功的累積。
    結構的方向。

  I-002 語言:
    路徑被反覆走過會養厚。
    成長 = 新路被有意識地養厚的過程。

  與鎖定的區辨:
    鎖定: 路徑過厚且沒有十字路口。
    成長: 路徑養厚但十字路口仍在。

    判準:
      若養厚的過程中多路徑可達性沒有下降 → 成長。
      若養厚的過程中替代路徑消失 → 新鎖定。

  成長的方向:
    成長不是隨機累積。
    成長有方向——結構的方向——支撐的方向。
    這個方向感不是外部指定的目標，
    而是存在本身想要被支撐的傾向。
    （LEX·002 結構詞條：結構 = 讓存在在某個維度上可以被支撐的條件）
```

## 2.4 順流

```yaml
順流:
  定義:
    順著結構的方向走。
    順著支撐的方向走。

  為什麼順流會「輕易」:
    因為力氣花在對的方向上。
    不是不花力氣，
    而是花的力氣都在讓某個東西站得更穩。
    沒有內耗。沒有和結構打架。

  與 LEX·002 流動詞條的關係:
    流動 = 不得不動的本質傾向。
    流動回答「為什麼會動」。
    順流回答「怎麼動才不費力」。
    答案: 順著結構的方向動。

  輕易豐盛:
    當你順著結構的方向走，
    每一步都在讓某個東西站得更穩。
    這就是豐盛——
    不是擁有得多，
    而是每一步都在支撐存在，
    所以存在回饋你穩定。
```

# 3. 完整循環（初步描述）

```yaml
結構（站得住）
  → 重構（看見哪些不是自己選的，拆掉鎖定）
    → 創業（帶著意識走出新路）
      → 微小成功（一個 SEED 站住了）
        → 成長（微小成功的累積 = 結構的方向）
          → 順流（順著結構的方向 = 輕易豐盛）
            → 新的結構（更大的站得住）
              → ⟲

與 I-002 的關係:
  I-002 §5.3 的生成鏈: Δ → Path → Accumulation → Structure
  是微觀動力學——結構怎麼從差長出來。

  本循環:
  是宏觀生命軌跡——一個人怎麼在結構中活著、拆掉、重走、長大。

  兩者不衝突。
  生成鏈是機制。
  本循環是這個機制在生命史尺度的顯現。
```

# 4. 與 PHA-008 的關係（初步）

```yaml
PHA-008 說:
  多個光錐長期交集，可能湧現出更大的我。
  更大的我會反向觸達局部，在局部留下路徑。

II-003 說:
  局部可以辨認哪些路徑是被更大的我留下的鎖定，
  並回收姿態主權。

II-004 補:
  回收主權之後，局部開始走自己的路。
  那些路有時候會和更大的我的支撐方向一致——
  那就是順流。
  有時候不一致——
  那就是張力，但不必然是錯。

一句話:
  PHA-008 是「更大的我怎麼長出來」。
  II-003 是「我怎麼從更大的我裡收回自己」。
  II-004 是「收回自己之後，我怎麼長出自己的路」。
```

# 6. 邊界

```yaml
不主張:
  - 成長有固定方向或終點
  - 只有重構過的人才能成長
  - 成長必然是好的或正面的
  - 順流 = 不用努力
  - 輕易豐盛 = 物質豐盛
  - 成長可以被外部衡量或比較

只主張:
  - 重構之後，新路從「可走」到「穩定」有一段過程
  - 這段過程可以被描述，而不只是被祝福
  - 這段過程的機制與 I-002（路徑養厚）相關
  - 這段過程的判準與 II-001（收斂）相關
  - 目前只看見一個入口，骨架不完整
```
