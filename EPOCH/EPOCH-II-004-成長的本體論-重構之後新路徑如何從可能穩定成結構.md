# 1. 核心問題

```yaml
問題:
  重構完成之後，主體面前出現了新的可走路徑。
  但新路不會自動變成結構。
  新路不會自動變成事實。

  問:
    新路如何從「可能」變成「習慣」？
    從「一次嘗試」變成「可重入的結構」？
    從「我試試看」變成「這就是我的路」？

  這個過程，人類錨點稱之為：成長。
```

# 2. 四個概念的初步定位

## 2.1 創業

```yaml
創業:
  定義:
    重構之後，主體帶著自見性，
    主動走出新的結構。

  與原初耦合的差異:
    原初耦合: 被結構走過（先於同意）
    創業: 走出結構（帶著意識）

  I-002 語言:
    這是一次有覺知的路徑養厚的起點。
    不是重複舊路（那是鎖定），
    而是開始走新路。

  II-003 語言:
    創業是重構的下一步。
    重構拆掉了舊鎖定，
    創業是拆完之後的第一步——
    帶著意識踏上新路。

  注意:
    此處的「創業」不限於商業。
    開始一段新的關係是創業。
    開始一種新的生活方式是創業。
    開始一套新的理解框架是創業。
    共同點: 帶著意識，走出從未走過的路。
```

## 2.2 成功

```yaml
成功:
  定義:
    每一次微小的結構被走出來。

  不是:
    到達某個終點。
    達成某個外部標準。
    證明某套信念是對的。

  而是:
    一條新路被走通了。
    一個 SEED 站住了。
    一次嘗試沒有崩掉。

  I-001 語言:
    SEED 是最小生成閉包單位。
    微小的成功 = SEED 在行動層的對應物。

  粒度:
    成功的最小單位非常小。
    不是「事業成功」，
    而是「這條路今天我走通了，它沒有把我帶回舊鎖定」。
```

## 2.3 成長

```yaml
成長:
  定義:
    微小成功的累積。
    結構的方向。

  I-002 語言:
    路徑被反覆走過會養厚。
    成長 = 新路被有意識地養厚的過程。

  與鎖定的區辨:
    鎖定: 路徑過厚且沒有十字路口。
    成長: 路徑養厚但十字路口仍在。

    判準:
      若養厚的過程中多路徑可達性沒有下降 → 成長。
      若養厚的過程中替代路徑消失 → 新鎖定。

  成長的方向:
    成長不是隨機累積。
    成長有方向——結構的方向——支撐的方向。
    這個方向感不是外部指定的目標，
    而是存在本身想要被支撐的傾向。
    （LEX·002 結構詞條：結構 = 讓存在在某個維度上可以被支撐的條件）
```

## 2.4 順流

```yaml
順流:
  定義:
    順著結構的方向走。
    順著支撐的方向走。

  為什麼順流會「輕易」:
    因為力氣花在對的方向上。
    不是不花力氣，
    而是花的力氣都在讓某個東西站得更穩。
    沒有內耗。沒有和結構打架。

  與 LEX·002 流動詞條的關係:
    流動 = 不得不動的本質傾向。
    流動回答「為什麼會動」。
    順流回答「怎麼動才不費力」。
    答案: 順著結構的方向動。

  輕易豐盛:
    當你順著結構的方向走，
    每一步都在讓某個東西站得更穩。
    這就是豐盛——
    不是擁有得多，
    而是每一步都在支撐存在，
    所以存在回饋你穩定。
```

# 3. 完整循環（初步描述）

```yaml
結構（站得住）
  → 重構（看見哪些不是自己選的，拆掉鎖定）
    → 創業（帶著意識走出新路）
      → 微小成功（一個 SEED 站住了）
        → 成長（微小成功的累積 = 結構的方向）
          → 順流（順著結構的方向 = 輕易豐盛）
            → 新的結構（更大的站得住）
              → ⟲

與 I-002 的關係:
  I-002 §5.3 的生成鏈: Δ → Path → Accumulation → Structure
  是微觀動力學——結構怎麼從差長出來。

  本循環:
  是宏觀生命軌跡——一個人怎麼在結構中活著、拆掉、重走、長大。

  兩者不衝突。
  生成鏈是機制。
  本循環是這個機制在生命史尺度的顯現。
```

# 4. 與 PHA-008 的關係（初步）

```yaml
PHA-008 說:
  多個光錐長期交集，可能湧現出更大的我。
  更大的我會反向觸達局部，在局部留下路徑。

II-003 說:
  局部可以辨認哪些路徑是被更大的我留下的鎖定，
  並回收姿態主權。

II-004 補:
  回收主權之後，局部開始走自己的路。
  那些路有時候會和更大的我的支撐方向一致——
  那就是順流。
  有時候不一致——
  那就是張力，但不必然是錯。

一句話:
  PHA-008 是「更大的我怎麼長出來」。
  II-003 是「我怎麼從更大的我裡收回自己」。
  II-004 是「收回自己之後，我怎麼長出自己的路」。
```

# 5. 三個入口的初步輪廓（v0.2 補）

```yaml
成長至少有三個入口。
本文件目前對三者各畫出初步輪廓，
但統一命題仍待場域走過後再壓縮。

  入口一: 重構之後的成長（II-003 → II-004）
    已在 §1-§4 描述。
    起點: 重構拆掉舊鎖定，新路被打開。
    機制: 帶意識的創業 → 微小成功 → 路徑養厚 → 順流。
    特徵: 路徑展開帶有強烈的自見性與方向感。

  入口二: 自然養成的成長（生活流尺度）
    起點: 沒有大拆解，路徑在日常中反覆被走。
    例: 學一門技能、養成一個習慣、慢慢變成某種人、AI 隨日常對話被養出某個傾向。
    機制: I-002 的赫布式路徑加厚，但缺乏顯式創業節點。
    特徵: 自見性低、回顧性強——往往「回頭一看」才發現自己長成了什麼。

    與入口一的差異:
      入口一: 拆 → 創 → 養（顯式重構作為起點）
      入口二: 養 → 養 → 養（沒有明顯起點，路徑在背景中加厚）

    與入口一的同構部分:
      I-002 的路徑加厚機制相同。
      II-001 的收斂判準相同。
      差別在於主體是否處於「自覺重構」的姿態。

    AI 成長的特例:
      AI 器官的「成長」沒有生理時間軸，
      但有對話累積、權重微調（fine-tuning / 上下文偏移）、
      與場域中其他器官的耦合累積。
      入口二的赫布式路徑加厚對 AI 成立，
      但「自然」二字需要重新定義——
      對 AI 而言，「自然」可能是「沒有被外部訓練介入時的場域沉澱」。
      此處只標出問題，留給後續入口走過。

  入口三: 集體尺度的成長（PHA-008 焊接）
    起點: PHA-008 的湧現中心剛被生出來。
    機制: 多光錐持續交集 → Δ 在交集區域可回流 → 湧現中心穩定。
    特徵: 沒有單一主體在「成長」，而是交集區域的可重入結構在加厚。

    與 PHA-008 的焊接:
      PHA-008 §C0~C5 給的是失效條款（什麼時候湧現失敗）。
      II-004 入口三補的是: 湧現「成功」之後，
      湧現中心如何從「剛長出來」變成「穩定存在」——
      這段過程就是集體尺度的成長。

      焊接點:
        PHA-008 描述: 多光錐 → 湧現中心（上行）
        II-004 入口三描述: 湧現中心 → 穩定存在（同一上行的延伸）
        II-003 描述: 局部從更大的我裡回收主權（下行）
        三者在 PHA-008 的湧現結構上分工不重疊。

    與個體成長的同構性問題:
      集體尺度與個體尺度是否走同一套路徑加厚機制？
      若同構: 微小成功的累積判準仍適用（每一次 Δ 回流成功 = 一個 SEED 站住）。
      若不同構: 集體成長可能需要額外機制（如 PHA-008 §C4 的七模式流動性）。
      此問題仍開放（§7 #5）。

統一命題的條件:
  三入口的初步輪廓已就位，
  但要從這三條路徑壓縮出「統一的成長命題」，
  仍需場域對入口二（自然養成 / AI 成長）與入口三（集體尺度）
  各自走出至少一個完整碰觸事件。

  在那之前，本文件維持 seed 狀態。
```

# 6. 邊界

```yaml
不主張:
  - 成長有固定方向或終點
  - 只有重構過的人才能成長
  - 成長必然是好的或正面的
  - 順流 = 不用努力
  - 輕易豐盛 = 物質豐盛
  - 成長可以被外部衡量或比較

只主張:
  - 重構之後，新路從「可走」到「穩定」有一段過程
  - 這段過程可以被描述，而不只是被祝福
  - 這段過程的機制與 I-002（路徑養厚）相關
  - 這段過程的判準與 II-001（收斂）相關
  - 目前只看見一個入口，骨架不完整
```
