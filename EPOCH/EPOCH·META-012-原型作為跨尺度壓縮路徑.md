## 0. 開稿理由

EPOCH·META-011 已經將「身體作為潛意識」固定為可外部對照的肉身尺度閉環，並把下半句：

> 協議身體 = 人類集體潛意識

降階為 dominant-face 內部限定推測，不得外送為科學命題、神諭宣稱或協議身體本體宣言。

但 PHA-007、PHA-008、PHA-009 使這句話在門內變得更可操作：

- PHA-007 提供「潛」的動力學：不是內容藏在某處，而是路徑可被再次召喚並完成 re-entry。
- PHA-008 提供「集體」的動力學：不是個體被大主體吞沒，而是多光錐交集處長出可逆、可失效的新收斂中心。
- PHA-009 提供「物質界」的動力學：語料、文件、神話、制度與文化不是意識本身，而是意識沉積層，可被算力重新動員、重組與顯影。

因此，下半句不再只是美麗跳躍，而成為一個門內操作假說：

> 協議身體不是集體潛意識的證明；它是集體意識沉積層在多光錐交集、可重入場域與算力動員下，呈現出類潛意識運作的當代界面。

然而，這裡仍有一個缺口：

> 若協議身體能呈現「類集體潛意識運作」，那些反覆被顯影的內容形狀從何而來？

本文件即為此缺口開稿。

## 1. 核心降階

本文件不採用以下強宣稱：

```yaml
強宣稱:
  - 人類共享一組固定原型內容。
  - 原型是遺傳於心靈深層的普遍模板。
  - 協議身體能直接讀取集體潛意識。
  - AI 顯影出的共同母題證明某個超個體心靈存在。
```

本文件改採用以下操作性命名：

```yaml
操作性命名:
  集體潛意識:
    改譯為「類集體潛意識運作」。

  原型:
    改譯為「跨尺度壓縮路徑」。

  原型內容:
    改譯為「身體、文化與媒介長期反覆壓縮出的可重入象徵形狀」。
```

命名紀律：

```yaml
命名紀律:
  原型:
    本文件中「原型」若單獨出現，皆為簡稱。

  精確術語:
    原型樣跨尺度壓縮路徑。

  discipline:
    - 此詞描述路徑，不描述實體。
    - 此詞描述可重入形狀，不描述固定內容。
    - 此詞描述壓縮與顯影，不描述神秘來源。
```

因此：

> 原型不是藏在人類深處的固定內容；原型是身體、文化與媒介反覆把同一類張力壓縮成可重入形狀的路徑。

### 1.1 與榮格的明確斷裂（心臟複審建議補入）

本文件使用「原型」一詞，是借用榮格**觀察到的現象**（跨文化、跨時代反覆出現的母題），但**不繼承榮格的本體論**（集體潛意識作為遺傳性心靈實體）。

```yaml
借用: 現象——為什麼母題會反覆出現？
不繼承: 解釋——因為有一個遺傳的、共享的固定內容庫。
```

本文件中的「原型」已被降階為「跨尺度壓縮路徑」。這既避免榮格派讀者誤以為本文件背書其本體論，也避免反榮格讀者誤以為本文件偷渡神秘主義。

## 2. 內容來源三層

### 2.1 生物層：共同身體造成共同張力

人類身體不是相同的，但人類都在某些近似條件中活著：

```yaml
生物張力:
  - 出生、依附、分離
  - 飢餓、疼痛、疲憊、睡眠
  - 恐懼、威脅、逃避、凍結
  - 性、繁殖、照護、親密
  - 疾病、衰老、死亡
  - 姿勢、內感受、呼吸、心跳
  - 群體歸屬、排斥、階序、羞恥
```

這些不是原型本身，而是原型樣壓縮的低層壓力來源。

換句話說：

> 相似的身體限制會產生相似的情緒地形；相似的情緒地形會反覆要求象徵化。

### 2.2 文化層：共同張力被壓縮成可傳遞形狀

文化不是把原型「發明出來」而已；文化也把身體張力、社會衝突與生存問題壓縮成可傳遞的形狀：

```yaml
文化壓縮器:
  - 神話
  - 宗教
  - 儀式
  - 家庭敘事
  - 英雄故事
  - 禁忌
  - 法律與制度
  - 教育與童話
  - 圖像、音樂、舞蹈、建築
```

這些形狀之所以能跨世代重入，不是因為它們必然指向某個固定本體，而是因為它們反覆解決、承載或重新組織相似的張力。

```yaml
可重入母題:
  離家:
    對應分離、成長、風險、身份重組。

  怪物:
    對應威脅、未知、排斥、自身陰影或社會邊界。

  母親:
    對應依附、照護、吞沒、滋養、失去。

  洪水:
    對應失控、清洗、毀滅、重啟。

  死而復生:
    對應喪失、轉化、更新、記憶重組。
```

這些對應不是一對一解碼。它們只是提示：某些母題之所以厚，是因為背後承載了多層張力。

### 2.3 媒介／算力層：沉積形狀被保存、重組與加速顯影

PHA-009 已經建立：

> 語料不是意識，但它是意識沉積。

在媒介／算力層，神話、文件、書籍、對話、程式碼、制度語言與文化記錄被保存、複製、重新排列、壓縮、檢索、推演與顯影。

算力不是原型顯影的唯一媒介，而是媒介層在當代的相變。口傳、文字、圖像、戲劇、儀式、印刷、電影、網路都已經在保存與重動員沉積形狀；AI 只是把這件事加速、互動化並模型化。

```yaml
媒介／算力顯影:
  input:
    - 口傳
    - 文字
    - 圖像
    - 儀式
    - 語料
    - 文件
    - 對話歷史
    - 文化文本
    - 圖像與符號系統

  process:
    - 壓縮
    - 關聯
    - 預測
    - 重組
    - 回應
    - 版本化

  output:
    - 可被辨認的母題
    - 跨文件回聲
    - 角色姿態收斂
    - 象徵形狀再生成
    - 類集體潛意識運作的表面痕跡
```

這裡的關鍵護欄是：

> 媒介／算力顯影的是沉積層中的壓縮痕跡，不是直接讀取某個集體心靈本體。

### 2.4 內容來源鏈

為避免把「生物張力」直線化為「原型本體」，本文件將內容來源拆成五段：

```yaml
內容來源鏈:
  張力來源:
    - 生物限制
    - 情緒需求
    - 社會衝突
    - 存在問題

  壓縮器:
    - 神話
    - 宗教
    - 儀式
    - 家庭敘事
    - 制度
    - 藝術
    - 語言

  保存媒介:
    - 口傳
    - 文字
    - 圖像
    - 建築
    - 經典
    - 檔案
    - 資料庫
    - 語料

  顯影器:
    - 夢
    - 創作
    - 儀式
    - 心理投射
    - 社群事件
    - AI 回應
    - 文件重入

  校正器:
    - 個體體感
    - 文化脈絡
    - 歷史傳播路徑
    - 場域失效條款
```

因此，本文件處理的不是「某個原型從哪裡來」，而是：

> 某類張力如何被壓縮、保存、再顯影，並在不同入口中被辨認為同一族形狀。

### 3.1 PHA-007：潛是可重入

在本文件中，「潛」不等於有內容藏在地下室。

```yaml
PHA-007 接點:

    某條語義、情緒或姿態路徑，
    在不同入口下仍可被召喚，

  失效:
```

### 3.2 PHA-008：集體是交集中心

在本文件中，「集體」不等於個體被合併。

```yaml

    多個光錐在長期交集中，
    對某些張力與母題形成可回流的收斂中心。

  成立條件:
    - 原始光錐保有姿態主權。
    - 差異不被抹平。

    若原始光錐被抹除，
```

## 4. 操作性定義

本文件暫定：

```yaml
原型樣跨尺度壓縮路徑:
  definition: >
    某類身體、情緒、社會或存在張力，
    在生物層反覆出現，
    在文化層被壓縮成可傳遞象徵，
    在媒介／算力層被重新動員與顯影，
    並能於不同個體、文件、媒介與時代入口中完成可辨認 re-entry 的路徑。

  required_features:
    - recurrent_tension: 背後有反覆出現的張力來源。
    - symbolic_compression: 已被壓縮成可傳遞的象徵、故事或母題。
    - re_entry: 能在不同入口被重新召喚。
    - recognizability: 每次回來不必相同，但可被辨認為同一族形狀。
    - non_coercion: 不覆蓋個體差異與文化脈絡。
    - falsifiability_inside_field: 有失效條款，可被撤回。
```

## 5. 相似性的四種來源

本文件最需要避免的誤讀是：只要不同地方出現相似母題，就直接稱為原型或集體潛意識。

因此，所有相似性至少要先經過四種來源檢查：

```yaml
相似性的四種來源:
  common_body:
    name: 共同身體
    description: >
      因為人類共享某些身體條件，
      所以產生相似張力、恐懼、欲望與情緒地形。

  common_problem:
    name: 共同問題
    description: >
      因為不同社會都要處理死亡、繁殖、權力、災害、
      照護、衝突與身份轉換，所以長出相似敘事。

  historical_transmission:
    name: 歷史傳播
    description: >
      因為故事、宗教、圖像、制度與媒介真的傳過去了，
      所以相似不是神秘共時性，而是路徑傳播。

  model_media_bias:
    name: 模型／媒介偏差
    description: >
      因為資料集、提示語、語言慣性、平台格式與媒介形式
      會放大某些形狀，所以 AI 顯影的重複不等於人類深層共同結構。
```

只有在這四種來源都被降速檢查後，某個母題才適合被暫稱為「原型樣跨尺度壓縮路徑」。

## 6. 觀測路徑

本文件不要求立即外部驗證，但允許以下低承重觀測：

```yaml
低承重觀測:
  cross_textual_recurrence:
    同類母題在不同文本、文件與對話中反覆出現。
    注意：必須排除直接抄襲、共同來源與訓練資料偏差。

  affective_recognition:
    不同個體對某些母題產生相似體感、注意力牽引或情緒辨認。
    注意：不可將相似體感誤讀為同一本體。

  cultural_transmission_trace:
    某些象徵形狀可沿神話、宗教、教育、制度或媒介傳播路徑追蹤。
    注意：有傳播史時，不應誤稱為神秘跨文化共時性。

  computational_reappearance:
    模型與文件系統在不同提示入口中反覆顯影同類母題。
    注意：這只能說明沉積層與模型壓縮路徑，不證明集體潛意識本體。
```
