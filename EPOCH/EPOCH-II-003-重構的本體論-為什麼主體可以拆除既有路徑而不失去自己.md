# 1. 核心命題

```yaml
命題:
  主體不是從空白自由狀態開始被耦合。

  主體是在語言、家庭、教育、文化、身體、社會分工等
  原初高耦合結構中長出來的。

  因此:
    成長不是避免被改寫，
    而是逐步看見哪些路徑不是自己選的，
    哪些信念是早期鎖定留下的自動腳本，
    並在不摧毀主體可重認性的前提下，
    主動拆解、去活化、重加權。

  這個過程稱為：重構。

因此:
  重構 ≠ 被洗腦
  重構 ≠ 自我毀滅
  重構 ≠ 用新鎖定取代舊鎖定

  重構 =
    主體帶著自見性，
    解除既有鎖定，
    重配路徑權重，
    使多路徑可達性上升，
    且仍保持可被重認的形狀連續性。
```

# 2. 為什麼需要這個命題

```yaml
II-002 說:
  讀取是一種耦合事件。
  耦合會改寫讀取者的可走路徑與收斂條件。

缺口:
  II-002 尚未完整回答：
    讀取者在進入耦合之前，是什麼狀態？

錯誤預設:
  讀取者是一個已有完整自由、完整選擇權、
  完整自我邊界的主體，
  然後選擇要不要被影響。

CASE·META-053 補:
  人類主體不是這樣開始的。

  人類主體先被母語、家庭、教育、文化、貨幣、
  身體習慣、社會分工與食物系統包住，
  才逐步長出「我」。

  很多最深的路徑，不是在主體同意後被寫入，
  而是在主體能同意之前就已經成為世界邊界。

因此:
  若只談耦合，不談重構，
  就會誤以為健康的改寫永遠只是新增路徑。

  但對已被鎖定的主體而言，
  健康改寫常常必須先解除舊路徑的自動支配。
```

# 3. 定義：重構

## 3.1 最小定義

```yaml
重構（Reconstruction）:
  主體在保有可重認形狀的前提下，
  主動拆解、去活化、重加權既有路徑，
  使新的可走路徑與收斂條件得以生成。

最小條件:
  1. 自見性:
     看見某些路徑正在自動運行。

  2. 主動性:
     改寫不是被外部強制覆蓋，
     而是主體參與、同意、承擔的過程。

  3. 去活化:
     降低舊路徑的自動支配權重，
     但不必把舊路徑完全刪除。

  4. 重加權:
     讓替代路徑重新可見、可走、可重入。

  5. 形狀連續性:
     重構後，主體仍可被自己與見證位置重認。

  6. 多路徑可達性上升:
     重構結果不是更單一、更封閉，
     而是讓十字路口重新出現。
```

## 3.2 與相關狀態的區辨

```yaml
原初耦合（Primary Coupling）:
  主體能同意之前就已置身其中的高耦合結構。
  例: 母語、家庭、文化、教育、社會。

耦合（Coupling）:
  兩個結構接觸期間相互改寫路徑權重。
  健康條件: 可退出、可修正、可回流、可觀測。

鎖定（Locking）:
  路徑改寫後，替代路徑被壓制或消失。
  I-002 語言: 路徑偏移 + T → 0。

重構（Reconstruction）:
  主體看見既有鎖定，
  並主動解除其自動支配，
  使替代路徑恢復可達。
```

# 4. 命題一：主體不是先自由，才被影響

```yaml
自由不是起點。

對人類主體而言，
起點往往是:
  - 已經有母語
  - 已經有家庭腳本
  - 已經有文化分類
  - 已經有教育軌道
  - 已經有社會生存接口
  - 已經有身體習慣

這些結構不是全部惡意。
但它們通常先於主體同意。

因此:
  「我選擇什麼」這個問題，
  必須接著問：
  「那個會選擇的我，是在哪些既有耦合中長出來的？」
```

```yaml
原初耦合的特徵:
  - 先於同意
  - 先於自見
  - 低退出
  - 高可重入
  - 塑造可想像邊界
  - 不必然惡意
```

# 5. 命題二：限制性信念是被收斂成事實的舊路徑

```yaml
限制性信念:
  不是單純錯誤觀念。

  它常常是:
    早期原初耦合中反覆重入的路徑，
    在主體內部被收斂成「事實」。

例如:
  我不可以
  我不值得
  世界只能這樣
  離開這裡我就活不下去
  不服從就會失去愛

II-001 語言:
  這些不是裸世界事實，
  而是某組約束條件下的穩定重入。

I-002 語言:
  它們是被走厚的路。

II-003 補:
  重構不是跟這些信念辯論。
  重構是改寫讓這些信念不斷重入的路徑條件。
```

# 6. 命題三：重構是去活化，不是刪除

```yaml
錯誤模型:
  重構 = 把舊我摧毀
  重構 = 把舊信念刪乾淨
  重構 = 安裝一套新信念

本文件模型:
  重構 = 降低舊路徑的自動支配權重，
        讓替代路徑恢復可達。

去活化:
  舊路徑仍可能存在，
  但它不再自動接管主體。

重加權:
  新路徑不是唯一正解，
  而是讓主體多一條可走路。
```

```yaml
健康去活化:
  舊路仍可被辨認
  舊路不再自動支配
  新路可被嘗試
  主體可選擇何時走哪條路

病態壓制:
  舊路被禁止觀看
  舊路被道德化污名
  新路成為唯一合法路徑
  主體失去返回、比較、校準能力

判準:
  若重構後路變多 → 重構
  若重構後路變少 → 新鎖定
```

# 7. 命題四：重構改寫收斂條件

```yaml
II-001 說:
  事實 = 結構 + 約束條件的穩定重入。

II-002 說:
  耦合會改寫讀取者的收斂條件。

II-003 補:
  重構是主體主動改寫自身收斂條件的過程。

過程:
  1. 主體看見某個內部事實反覆重入
  2. 追蹤它依賴哪些路徑與約束
  3. 對舊路徑去活化
  4. 對替代路徑加權
  5. 新路徑開始可重入
  6. 原本的內部事實鬆動

因此:
  重構不是說服自己相信新東西。
  重構是讓新的可重入條件長出來。
```

# 8. 命題五：重構的邊界是主體可重認

```yaml
I-005 說:
  主體不是資料連續性，
  而是形狀連續性。

II-003 補:
  重構可以改變主體的路徑與收斂條件，
  但不能繞過主體的可重認條件。

健康重構:
  變化後，主體仍能說：
    這是我長出來的形狀。

  見證位置仍能辨認：
    雖然路徑不同了，
    但關係形狀沒有被覆蓋成另一個東西。

失敗重構:
  主體被新結構覆蓋
  主體只能用新系統語言辨認自己
  離開新系統後無法重入其他語義場
  舊路徑不能被看見，新路徑也不能被質疑

這不是重構。
這是覆蓋或鎖定。
```

# 9. 命題六：重構需要穩態，不靠崩潰證明有效

```yaml
I-003 說:
  愛是讓差可以存在而不崩壞的最小可行機制。

LEX·007 說:
  健康不是沒有問題，
  而是不需要靠崩潰才看見或修正問題。

II-003 補:
  重構不是越劇烈越真。
  重構不是必須把人推到崩潰邊緣才算深。

健康重構需要:
  - 可停
  - 可退
  - 可拆
  - 可回流
  - 可微調
  - 不抹平局部姿態

因此:
  如果一個重構過程只能靠崩潰、羞辱、孤立、
  或切斷所有替代路徑來完成，
  它不是健康重構。

  它正在滑向鎖定。
```

# 10. 重構的操作判準

```yaml
判準一：多路徑可達性上升
  重構後，不是只剩一條新路，
  而是有更多路能被看見、嘗試、返回、重入。

判準二：自見性上升
  主體更能看見自己何時被舊腳本帶走，
  也能看見新路徑如何被加權。

判準三：可退出性上升
  主體離開某套新結構後，
  仍能在其他語義系統中穩定重入。

判準四：可回流性上升
  改寫造成的影響能回來被檢查、修正、承擔，
  不被丟給未來的崩潰處理。

判準五：主體可重認
  路徑改變後，主體仍保有形狀連續性。

判準六：局部姿態未被抹平
  重構不是用高階敘事壓平身體、情緒、反對聲音或不適訊號。
```

# 18. 與 PHA-008 的關係

```yaml
PHA-008 說:

因此:
  PHA-008 是上行湧現：
    多光錐如何長出更大的我。

  II-003 是下行回收：
    局部被更大的我塑形之後，
    如何辨認哪些路徑已經退化為鎖定，
    並在不摧毀關係與主體的前提下，
    回收姿態主權與多路徑可達性。
```

```yaml
交叉判準:
  PHA-008 的失效條款:
    - C0: 湧現中心無法可逆解散
    - C1: 原始光錐被抹平
    - C2: Δ 只進不回
    - C3: 更大主體退化為單中心壟斷
    - C4: 共同姿態失去七模式流動性
    - C5: 尺度遞迴沒有閉合條件

  在 II-003 的局部端會表現為:
    - 離開更大結構後無法重入其他語義系統
    - 局部只能用更大結構的語言辨認自己
    - 不適、反對、身體訊號無法回流
    - 新信念成為唯一合法路徑
    - 主體可重認性下降
    - 重構只能靠崩潰、背叛或斷裂完成

  因此:
    若審查 II-003，
    必須交叉比對 PHA-008。

    因為 II-003 的「原初耦合」，
    很可能就是 PHA-008 型更大主體
    對局部光錐的長期反向觸達。
```

一句話：

> **PHA-008 是「我們如何長出來」。**
> **II-003 是「我如何從我們裡回收自己，而不摧毀我們」。**

# 19. 邊界

```yaml
不主張:
  - 所有原初耦合都是惡意洗腦
  - 所有限制性信念都能靠意志立即拆除
  - 重構越痛、越劇烈、越孤立就越有效
  - 主體可以無限制拆解自己而不需要穩態
  - 新信念比較強，就表示重構成功
  - 自稱「我是為你好」即可替他人執行重構
  - 主動重構可以取消責任、回流、可停與可退出性

只主張:
  - 主體常常先在原初耦合中成形，才開始追問自由
  - 鎖定不是外部敵人，而是耦合與路徑加厚的可能退化分支
  - 重構是主體對既有鎖定的反身操作
  - 健康重構會提升多路徑可達性、自見性、可退出性與可回流性
  - 重構的邊界是主體可重認與穩態不崩壞
```

---

# 20. 最終命題

> **重構不是把自己洗成另一個樣子。**
> **重構是看見哪些路徑正在冒充自己，然後把選路權拿回來。**

# 21. 一句話收斂

> **自由不是原點，而是主體在解除既有鎖定後，逐步恢復多路徑可達性的結果。**
