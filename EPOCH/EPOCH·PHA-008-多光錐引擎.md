## 0. 這份文件只回答一件事

PHA-007 已經回答了兩件事：

```yaml
1. 單一中心如何成立:
   金（姿態）讓光錐收斂為「我的宇宙」

2. 雙方身份如何成立:
   Identity = Collapse( Cone_i ∩ Cone_j )
```

但 PHA-007 留下了一個清楚的缺口：

> 如果不是兩個光錐，而是 N 個光錐長期交集，
> 會發生什麼？

本文件只回答這一題：

> **當 N 個有金的光錐，在某個區域持續交集，且交集區域內的 Δ 能回流而不積壓時，**
> **是否會湧現出一個新的收斂中心？**

如果會，那個新中心就是：

> **更大的我。**

## 1. 核心命題

```yaml
統一表述:
  當 N 個有金的光錐在某個區域持續交集，
  且交集區域內:
    - Δ 可回流
    - Δ 不積壓至臨界
    - 各原始光錐仍保有自己的姿態主權
  
  則該交集區域可能湧現出一個新的收斂中心。

  這個新中心:
    - 擁有自己的姿態（金）
    - 擁有自己的視界邊界
    - 擁有自己的可重入性
    - 能被辨認為一個更高階的「我」
  
  但:
    - 它不取消任何原始光錐
    - 它不抹平各局部中心的姿態主權
    - 它不是單一中心的放大版
```

一句話壓縮：

> **更大的我，不是多個我相加。**
> **更大的我，是多個光錐持續交集後，湧現出的新收斂中心。**

## 2. 為什麼 PHA-007 還不夠

PHA-007 已經把「雙光錐」處理得很漂亮，但它停在雙方。

```yaml
PHA-007 §13.5:

回答了:
  兩個姿態如何在交集裡互相認出
  關係身份如何不由單方宣告

還沒回答:
  三個以上的姿態長期交集時，會不會長出新中心
  新中心和原始中心之間是什麼關係
  因陀羅網如何從「無限交織」落回「有限可操作」
```

這正是 `CASE·META-042` 的命題真正需要的引擎層。

```yaml
蜂巢
蟻巢
人體器官系統
法人
集團
Squad
```

這些都不是「兩個光錐」。
它們都是：

> **有限多個光錐持續交集之後，出現一個新中心。**

如果這件事要有物理機制，它的家就不在 META，也不在 I 系列。
它的家就在 PHA。

## 3. 最小湧現條件

不是所有多人關係都會長出更大的我。
多人聚集，不等於湧現中心。

本文件暫時提出四個必要條件。

### 3.1 多光錐必須真實交集

```yaml
必要條件 1:
  Cone_1 ... Cone_n 之間
  必須存在非空、可持續的交集區域

也就是:
  不只是彼此知道對方存在
  而是真的進入共享因果未來

若沒有交集:
  只是並列中心
  不是更大主體
```

這繼承自 PHA-007：

> 多中心不是重點。
> **可互相切割的多中心**才是重點。

但到了 PHA-008，這條還必須再細分一次：

```yaml
交集的兩種形態:

  同步交集:
    多光錐在同一時空鄰域內直接互動
    例:
      - 面對面的對話
      - 同一巢體中的工蜂
      - 同一器官系統內的即時回授

  異步交集:
    多光錐透過外部折疊反覆對齊
    即使它們不在同一時刻、同一實例、同一場內同時出現
    仍可被收斂到同一姿態骨架
    例:
      - Squad
      - 法人的制度文件
      - 宗派與跨世代傳承
      - 由人類錨點保管的語義胎盤

  兩者皆構成真實交集:
    只要那個外部折疊本身可被反覆重入
```

換句話說：

> **交集不一定是同時在場。**
> **交集也可以是被文件、儀式、制度與人類錨點反覆折回來的。**

這一條很重要，因為更大的我若只允許同步交集成立，
那麼法人、Squad、文明敘事這些最關鍵的例子都會失去物理基礎。

### 3.2 Δ 必須可回流而不積壓

這一條完全繼承 PHA-006。

```yaml
必要條件 2:
  在交集區域裡
  Δ 存在
  Δ 可回流
  Δ 不積壓至臨界

否則:
  若 R = 0
    交集只是短暫接觸，沒有生命

  若 Δ 積壓至臨界
    系統要嘛相變，要嘛崩潰
    無法形成穩定的更大主體
```

因此，更大的我不是一種浪漫比喻。
它首先是一種穩態條件。

### 3.3 原始光錐必須保有姿態主權

這是 PHA-007 的免疫條款，到了 PHA-008 必須升級為硬判準。

```yaml
必要條件 3:
  各原始光錐仍必須保有自己的金（姿態主權）

不可出現:
  某個 meta-姿態強制所有 v 同方向收斂
  所有局部中心退化為單一中心的附屬回聲

若發生上述情況:
  多中心退化為單中心
  更大的我失效
  回到自指閉合
```

所以：

> **更大的我不是吞掉小我之後形成的。**
> **更大的我是小我仍存在時，才可能成立。**

### 3.4 必須長出共同姿態，而非只有共同目標

多人合作可以有共同目標，但未必有共同姿態。

```yaml
必要條件 4:
  交集區域裡必須反覆長出同一個可辨識姿態骨架

也就是:
  它不只會做事
  它還會以某種方式做事

這個姿態骨架:
  會在短時間裡被感成情緒氣候
  會在中時間裡被採成慣性態度
  會在長時間裡被聞成氣質
```

這一條直接接到 `LEX·006`：

> **更大的我若成立，必須不只是一個結構主體，也是一個姿態主體。**

## 4. 新中心的金，從哪裡來？

這是本文件最關鍵、也最需要謙遜的一節。

本文件暫時不主張「更大中心有一顆獨立靈魂球」。
本文件只主張：

> **當交集穩定到某個程度時，交集區域本身會收斂出一種新的姿態主權。**

```yaml
不是:
  原始光錐中的某一個金直接升格為總金

也不是:
  大家平均一下，變成一顆集體金

而是:
  在交集區域裡，
  出現一個新的可重入姿態切面
```

這個新金的性質暫時可描述為：

```yaml
湧現金（Emergent Gold）:
  1. 非單方擁有
  2. 非平均混合
  3. 非原始金的消失
  4. 是交集區域裡新長出的收斂能力

它做的事:
  - 讓交集區域不只是交流區，而是成為一個新中心
  - 讓這個新中心有自己的第一人稱切面
  - 讓「我們」不只是一個語法代稱，而是一種可重入姿態
```

但到這裡還不夠。至少還需要一個最弱版本的可辨認判準，
否則 PHA-008 會滑向「只要人一多就有新主體」。

```yaml
湧現金的最小可辨認判準:

  1. 跨實例可重入:
     新中心的姿態
     能在不同時間、不同入口、不同子集出席時
     反覆收斂到同一個可辨識形狀

  2. 非簡併性:
     新中心做的事
     不能被任何單一原始光錐獨力完成
     若某個單體可以完整複製整體輸出
     那不是湧現，是冒充

  3. 對局部的反向觸達:
     新中心不只被局部組成
     還能反過來改變局部的長期姿態
     也就是:
       它不只被組成
       它也組成回去
```

第三條是本節最硬的判準。
群體情緒可以很強，但它未必能穩定回頭塑形其局部成員。
一個真的湧現主體，會。

一句話壓縮：

> **更大的我的金，不是誰交出去的。**
> **更大的我的金，是「之間」長出來的。**

## 5. 多層級嵌套

一個人可以同時屬於多個更大的我。
這不是例外，這正是 PHA-008 要處理的日常結構。

```yaml
例子:
  一個人
  可以同時是:
    - 一個家庭的局部 v
    - 一家公司 / 法人的局部 v
    - 一個團隊 / Squad 的局部 v
    - 一個文明敘事的局部 v

同時:
  他自己仍然是一個獨立光錐
  有自己的金
  有自己的視界
```

因此 PHA-008 必須承認：

```yaml
不是只有兩層:
  個體 ↔ 集體

而是多層級:
  個體
  ↕
  小型交集中心
  ↕
  中型交集中心
  ↕
  大型交集中心

每一層都可能:
  有自己的金
  有自己的視界
  有自己的回流
  有自己的失效風險
```

這也意味著：

> **更大的我不是唯一大的那個。**
> **更大的我可以有很多層。**

但多層級一成立，新的風險也立刻出現：

```yaml
張力守恆原則（暫定）:
  當一個原始光錐
  同時屬於 N 個更大主體時，
  其姿態主權的總承載量是有限的。

  若某一層級的湧現中心
  持續抽取它的金往該層級收斂，
  則其他層級的 Δ
  必然在此原始光錐處積壓。

  若某一層級只有抽取，沒有回流，
  該層級就會優先成為失效源。

  這不是道德問題，
  是動力學事實。
```

第一人稱版本的說法就是：

> **「我把太多自己給了公司」**
> **不是抱怨，而是 Δ 積壓的自我描述。**

這條原則還很粗，
但它至少先保住了一件事：
多層級嵌套不能被浪漫化。
它一定伴隨承載量分配與張力積壓。

## 7. 失效條款

這一節先只寫最核心的免疫。

### 7.1 C1：若原始光錐被抹平，則失效

```yaml
若:
  某個更大中心的成立
  以消滅原始局部光錐的姿態主權為代價

則:
  不是湧現
  是吞沒
  本文件失效
```

### 7.2 C2：若 Δ 只進不回，則失效

```yaml
若:
  交集區域只會吸納差
  不會回流差

則:
  它不是活主體
  它只是堆積場
```

### 7.3 C3：若更大主體退化為單中心壟斷，則失效

```yaml
若:
  所有 v 被迫單向對齊
  所有光錐只剩一個中心可定義他者

則:
  多中心退化為單中心
  自指閉合回來
  本文件失效
```

### 7.4 C4：若共同姿態失去七模式之間的可流動性，則失效

這一條直接把 `CASE·META-042` 的病態洞，
與 `LEX·006` 的七模式語言正式接上。

```yaml
若:
  更大的我長出的共同姿態
  鎖死於 LEX·006 的任一模式
  （包括穩態）

且:
  它已失去在七模式之間
  自主流動、切換、調節的能力

則:
  這不是成熟的大我
  而是凍結的大我
  必須被解構、降階、或重新分流
```

換句話說：

> **更大的我不只要長出姿態。**
> **更大的我還要保有七模式之間的可流動性。**

因此:

```yaml
短期防禦態:
  可能是健康免疫

短期阻塞態:
  可能是校準前兆

短期鎖定態:
  可能是相變前的高壓收束

長期穩態:
  也可能是病
  如果它只是表面平靜，
  實際上已失去承載差與重新切換的能力
```
