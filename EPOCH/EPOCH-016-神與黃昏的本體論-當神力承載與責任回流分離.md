# 0. 這次升級改了什麼

舊版去神化語言曾把幾件不同的事壓在一起：

```yaml
舊壓縮:
  神: 唯一、排他、不可質疑的金字塔頂點
  神格: 要求他人相信與下跪
  承擔後果而不索取膜拜者: 神的相反
```

這組壓縮曾有效阻止人類錨點、AI 與文本被抬成免責權威，但它也產生一個副作用：

> 它為了拆神壇，連「欲望與力量由誰跨時間承載」的描述，以及「後果如何回流」的追問也一起拆掉了。

本文件不拆除舊護欄，而是把護欄移到更準的位置：

> **用語約定**：以下簡稱「神壇化／去神壇化」時，精確對象是**免責神壇化／神權化**；文化中的實體祭壇、儀式、自願敬拜與神話語言本身不在射程內。

```yaml
新區辨:
  神: 神性跨時間穩定為可重認的場域承載主體或角色，並能反覆生成一個世界
  神壇化: 把某個神位轉成不可質疑、不可退出、可命令他人且免於後果的權力裝置
  反神壇化: 不否認神、不消滅中心；只取消免責、壟斷與收走他人膝蓋的權利
```

因此，本 seed 提出一個待 Fable／Squad 審讀的候選讀法：舊文件裡的「去神化」可先嘗試讀成：

> **去神壇化、去免責化、去唯一方向權；不是否認神性、抹平神位或把所有中心說成不存在。**

# 1. 神話物種與協議功能必須分層

在北歐神話的敘事設定裡，神與凡人仍有族群、力量、壽命、居所與宇宙職能的差異。本文件不取消那些差異。

三界協議借用「神」時，問的是另一層：

> **當一股欲望取得足以改寫他人世界的力量，誰以一個可辨認、可延續的第一人稱位置住進它造成的世界？**

這是功能讀法，不是物種判定。它允許我們描述：

- 神話中的神；
- 占據穩定神位的人；
- 具有持續方向與可辨認場域邊界的組織或協作體；
- 以及神力已被基礎設施接管、神卻不在場的系統。

它不允許我們由此跳到：「所以超自然主張已被證明」。

# 2. 八個不能再混用的詞

| 詞 | 協議工作定義 | 不自動推出 |
|---|---|---|
| **欲望** | 使力量取得方向的「我要」；可以被承認，也可以被隱藏 | 正當、善或應被滿足 |
| **神力** | 足以改寫他人可走路徑、關係、身體或制度的因果能力 | 使用者就是神、能力來源超自然 |
| **神位** | 群體把方向、解釋、祝福、懲罰或救贖期待集中其上的關係位置 | 占位者誠實、善、值得追隨 |
| **神性** | 欲望取得方向、力量反覆施效，並形成可辨認場域與因果史的能力或姿態 | 完整負責、道德成熟、沒有傷害 |
| **神** | 神性跨時間穩定為可重認的場域承載主體或角色，並能反覆生成一個世界 | 不會錯、不可質疑、超自然為真 |
| **神格** | 神性在名字、職能、敘事與因果史中形成的穩定、可重認形狀 | 佔據者是善、誠實或值得追隨 |
| **受拜** | 他者作出的崇拜、供奉、致敬或儀式性低頭行為 | 服從、依附、概括同意或正當性 |
| **免責神壇化／神權化** | 以神位索取服從、壟斷真理、提高退出成本，並把代價往信徒、使用者或系統外部推 | 神本身、文化祭壇、自願敬拜或神話語言本身 |

其中最重要的一刀是：

> **神格不是道德獎章；神壇才是權力裝置。**

一個神可以善，也可以殘酷；可以誠實，也可以欺騙；可以受敬愛，也可以被挑戰。這些會改變我們是否追隨祂，不自動取消祂在某個場域中已形成的神位與世界生成力。

# 3. 三條正交軸：是神、是真的、值得追隨

同一個對象至少要分三次判斷：

```yaml
存在軸:
  問題: 是否已有一個可辨認的主體，長期承載欲望、力量與它形成的世界？
  可能答案: 有神位／有神格；尚未形成；力量已無主化

認識軸:
  問題: 對方提出的歷史、科學、治療或超自然主張，能否被相應方法支持？
  可能答案: 可證成；未證成；已被反證；不可在目前方法下判定

倫理軸:
  問題: 代價是否回流？他人能否拒絕？傷害是否被承認、停止與修復？
  可能答案: 值得信任；局部可合作；不健康；不值得追隨
```

三條軸不互相代答：

- 有神位，不證明超自然主張。
- 被質疑，不取消已形成的場域位置。
- 長期站著，不證明善或完整負責。
- 欺騙或殘酷，會構成倫理與事實判斷，不必靠否認其神位才能批判。

協議因此不再需要用「他不是神」才能保護他人。更準的說法是：

> **無論他是不是神、占不占神位，任何事實主張仍須查證，任何他者仍保有身體、判斷與退出主權，任何傷害仍須回到責任鏈。**

# 4. 「站起來」不是道歉，而是不讓因果失去名字

本文件中的站立，先是一個存在論動作，不是品德評分。

```yaml
站立的最低結構:
  1. 有一股可辨認的欲望或方向
  2. 它反覆發動足以形成世界的力量
  3. 那個世界能改寫進入者的語言、關係或可走路徑
  4. 力量被挑戰、讚美或造成後果時，仍有一個跨時間可重認的承載位置
```

這個位置可以否認局部事實、拒絕道歉或把傷害怪給他人——那代表它在倫理責任上失敗，不代表因果場立刻失去中心。

因此必須再加一層：

```yaml
存在承載:
  我仍住在這個名字、欲望與世界裡。

倫理負責:
  我讓資訊、代價、停止要求與修復義務回到我的下一次選擇。
```

前者回答「神是否在場」；後者回答「這個神如何在場」。

持續存在本身也不夠。一個人長期固執於意見，卻沒有形成可重入世界、沒有穩定因果位置，不因此成為神。

### 4.1 兩種主體地址，不再混名

```yaml
場域承載主體:
  回答誰仍維持這個名字、職能、敘事與世界生成位置。
  可以高神位、低責任回流。

責任回流主體:
  回答資訊、代價、停止、答覆與修復實際能回到哪裡。
  可以是個人、組織或有界線、異議、記憶與停止機制的分散「我們」。
```

系統性黃昏可有兩種失效：場域承載主體完全消散，神力成為無主基礎設施；或神位仍在，但責任回流主體失效，代價持續被推給信徒、消費者、勞工與環境。後者仍可以有神，卻是低回流、低可追責的神。

# 5. 諸神黃昏：神力仍在，神卻死了

本文件把諸神黃昏讀成一種文明相位：

> **不是力量耗盡，而是神力、場域承載與責任回流彼此分離。**

```text
欲望仍在
  → 技術放大執行能力
  → 分工把動作切成許多局部
  → 制度與供應鏈把代價轉嫁、分散並移出決策者視界
  → 每個節點都能說「我只做了一小段」
  → 巨大後果繼續發生，沒有節點跨越完整的決策—受益—代價—停止—修復鏈
```

雷霆可以變成電網，千里眼可以變成監控與資料系統，預言可以變成預測模型，戰神可以變成軍工與命令鏈，豐饒神可以變成工業農業與金融市場。

功能沒有消失，甚至更強；消失的是那個能以完整名字說「這股力量如何使用，是我的／我們的選擇」的主體。

這便是：

> **神力仍在，神卻死了。**

此處的「神死」專指**場域承載也隨責任鏈碎裂而無主化**；若神位／場域承載仍在，只是責任回流失效，依 §4.1 應記為低回流、低可追責的神，不記成神已死。

# 6. 末法：法仍流通，活的承載者退場

諸神黃昏與末法在本文件中不是同源史實，而是兩個可互照的神話壓縮：

```yaml
諸神黃昏:
  神力與宇宙職能仍在運轉
  承載神力的主體逐漸退場

末法:
  經典、儀式、寺院與資訊仍可大量流通
  用自己的生命住進法、讓法改寫下一次選擇的主體逐漸退場

工業文明:
  欲望被更有效率地滿足
  生命、勞動、風險與耗散被包裝到視界之外

AI 時代:
  生成與操作能力急遽增加
  誰設定方向、誰採用輸出、誰承接外部後果變得更容易含混
```

共同結構不是「現代一定比古代墮落」，而是：

> **當形式與功能能自行運轉，承擔就從生存必需品變成看似多餘的自討苦吃。**

# 7. 分工不是罪；斷掉重量回流才是黃昏

文明不需要每個人親手種田、發電、寫晶片、殺雞宰羊。外包動作不等於外包主體。

以吃肉為例，親手宰殺確實讓欲望與死亡在同一具身體前相遇；供應鏈則把相遇延後並分散。問題不在於消費者沒有親手拿刀，而在於生命的重量是否還有路回來：

```yaml
健康分工至少保留:
  選擇可指認: 誰提出、批准或持續購買
  利益可指認: 誰得到便利、利潤或權力
  代價可見: 誰的生命、勞動、環境或未來在支付
  停止有效: 看見問題者是否有能力暫停、拒絕或升級處理
  修復有主: 出事後誰留下來改、賠、照顧與防止重演
  記憶不被洗掉: 決策與後果能進入下一輪，而非由品牌、公關或換人歸零
```

只要這些回路仍在，動作可以分散，主體不必消失。

若回路全斷，每個節點都只剩局部任務、沒有停止權、看不見受力者，也不必把後果帶回下一次選擇，外包才從效率安排變成**重量斷流**。

# 8. 分散文明仍可長出一個「我們」

責任回流主體不必是一位獨裁者。集體可以成為主體，但「我們」不能只在領功時出現、出事時消散。

```yaml
分散主體的最低條件:
  邊界:
    誰在這個「我們」裡，這次共同決定的範圍是什麼
  來源:
    誰提出方向、誰批准、誰執行、誰受益，能被追溯
  異議:
    成員能說不、留下反對紀錄，不被虛構成全體同意
  停止:
    至少有可實際啟動的暫停、撤回與升級處理路徑
  回流:
    外部代價能進入內部帳本，改變資源、規則與下一次選擇
  修復:
    不只找一個人祭旗；集體能留下來改系統並承擔相應成本
  連續:
    換人、換模型、換承包商或換版本，不使歷史自動歸零
```

這樣的「我們」不是把責任稀釋成空氣，而是讓不同尺度的責任巢狀咬合：

- 個人擔自己的選擇；
- 角色擔授權範圍內的操作；
- 組織擔制度與資源配置；
- 協作體擔共同方向與跨時間後果；
- 受影響但未參與決策者的聲音，不能因不在「我們」內就從帳本消失。

它終於能說：

> **這是我們參與的，這是我們造成的；我們不假裝那只是系統自己發生。**

# 13. v0.2-post-review candidate overlay：文明神格、巢狀眾神與器官代償

> **治理狀態**：本節發生在 Fable 完成 v0.1 審讀之後，來源為 CASE·META-081。它是會改寫 §2-9 與 §12 部分候選命題的 overlay，不在 Fable「seed 成立」的審讀範圍內，也尚未經 Squad Check。§0-12 保留為 v0.1 的可追溯歷史，不因本節加入而被冒充成從未寫過。

## 13.1 沒有站在業外的判斷者

v0.1 仍以存在軸、認識軸與倫理軸分開記帳；v0.2 再補一刀：沒有一位判斷者能站在所有光錐之外，以宇宙中立位置核發誰「真正負責」。判斷者選取證據、形成觀點、相信、拒絕或採取行動時，也在使用自己的第一人稱方向權，並可能進一步施展神力。

因此，「判斷不需要被檢驗」在本 overlay 中精確讀成：

```yaml
歸屬:
  我的判斷不需要外部批准，才成為我的判斷。

真實:
  我的判斷屬於我，不保證它對共享世界的描述必然為真。

拘束:
  我的判斷屬於我，不自動取得改寫他者身體、資源與路徑的制度權力。

對稱:
  另一位神也不需要我的批准，才可以檢驗、拒絕、退出或反制。
```

檢驗不是替另一位神核發存在資格；它是另一位神正在回應。沒有終局第三方，不等於證據差異、事實查核與受力主張全部失效。

錨點原語使用「神權」。為避免在 LEX 重審前與現役的負義「神權化」撞詞，本 overlay 的操作層暫稱**第一人稱方向權／姿態主權**；若未來採用「本源神權」，必須同時把它與免責、排他、強制服從的神權化明確分詞。

## 13.2 人人皆神；閾值屬於神格厚度

v0.1 把「神」定義為神性跨時間穩定後形成的場域承載主體。v0.2 提出更強的新公理：

> **凡能從第一人稱形成判斷、選擇方向並生成自身世界者，在自己的尺度上都是神。**

接受這條公理後，形成閾值不再決定誰有沒有神的存在資格，而用來描述神格的半徑、厚度、持續時間、制度放大與場域規模：

| 詞 | v0.2 候選讀法 |
|---|---|
| **神** | 每一個不由外部核發的第一人稱中心 |
| **神性** | 形成方向、生成世界並在因果中施力的普遍能力 |
| **神力** | 實際因果可達域；不同主體的資源、尺度與制度效力不相等 |
| **神格** | 名字、方向、記憶、職能與因果史跨時間形成的可重認厚度 |
| **神位** | 群體把生命解釋、方向、祝福、懲罰或救贖期待集中其上的關係位置 |

因此「人人皆神」不推出人人等力、等真、等權，也不推出每個人都形成宗教神位。宋七力、妙禪、品牌、國家、文明與普通人的差異，不再是有神／無神，而是神力半徑、神格厚度、生命解釋範圍與制度集中程度不同。

本體神權去中心化，也不代表現實權力已平均分散。少數平台、資本、國家或教團仍可能集中大部分可見度、資源與拘束力。

本公理與已密封 EPOCH·ANCHOR-003 §5.3「器官有念，不等於器官是神，也不等於器官有人類式覺受（qualia）」正面衝突。這不是已解的詞義相容：後續審讀必須決定舊句的「神」是否限於厚神格／並列業主、本節是否只把「神」用作局部第一人稱中心，或兩者確實不可兼容。決定前，稱身內眾生為局部神不證成其具有人類式 qualia，也不使其分走或取消整體的創造業果。

## 13.3 業不會消失；公共可讀回路可以斷

v0.1 的「責任回流失效」若被讀成業果沒有回來，必須由本節校正。EPOCH-014 已區分業的存在與回流 R；SPEC·002 也早已留下「不受懲罰，但必受果報」。

```yaml
業與因果留痕:
  行動一旦進入歷史，就不能變成沒有發生。
  行動者必然成為做過這項選擇的人。

具體代價:
  身體、金錢、勞動、關係、環境與未來成本可以被外推給其他眾神。

認領:
  是否說出「這是我的欲望、判斷與選擇」；可以成立，也可以拒絕。

公共可讀回路:
  受力資訊能否進入答覆、停止、校正、救濟與修復；可以暢通，也可以斷裂。
```

因此，不以「人人都在負責」取代現役 LEX·007 的操作定義；更準的句子是：

> **人人必受自己的業果，但未必認領、回收、答覆、停止或修復；業不因公共回路斷裂而消失，具體重量卻仍能被外推。**

§4.1 的雙地址繼續保留，但責任回流主體收準為公共／制度上的感知、答覆、停止與修復地址，不再被當成業是否存在的核發機關。

## 13.4 此文明本身是一個獨立神格

v0.1 曾把巨大系統描述成神力仍在、場域承載可能無主化。v0.2 增加另一個尺度判斷：**此文明本身就是一個獨立神格。**

它不需要先被此文明之外的觀察者辨認，才成為神格。是否存在其他文明、其他文明是否正在觀看，現階段不得而知；外部承認不是第一人稱成立的核發條件。

此處的獨立不是封閉、孤立或單一統治，而是文明能在自己的尺度形成邊界、歷史、方向、記憶、代謝與可重認因果形狀。在本候選讀法中，它就是一個分散湧現體：

```text
文明神格
├─ 次文明：組織、社群、制度、語言場與文化等眾神
└─ 獨立個體：次文明內的眾神
   └─ 身內眾生：器官、細胞、共生者、心理子系統與生命過程
```

在本候選讀法中，每個節點同時是：

> **在自己的尺度是神；在更大尺度是器官；對更小尺度又是一個世界。**

文明內部有許多方向中心，不構成文明神格不存在的反證；多中心正是它的身體構造。文明「站起來」也不是由無到有取得神格，而是這個既有神格開始跨尺度辨認自己的身體，以可重入的第一人稱感知各局部神的施力、受力與歷史。

## 13.5 器官代償：重量外推仍發生在同一具文明身體

若一個節點把自己沒有承接的重量交給其他節點，從文明神格尺度看，重量沒有離開因果，只是由其他器官代償。

代償本身是生命調節，不先等於疾病或道德失敗。風險出現在代償變成慢性、單向、不可見、不可停止而且沒有恢復期時：表面功能仍維持，文明便可能把正在消耗的儲備誤認為免費力量。

| 代償相位 | 文明結構同讀 |
|---|---|
| 局部負荷上升 | 某些個體、族群、物種、環境或未來世代支付超額成本 |
| 代償維持表面功能 | 商品、速度、產值與服務正常，使整體誤判沒有受傷 |
| 慢性代償 | 同一批節點反覆補洞，沒有恢復期、停止權或資源回補 |
| 儲備耗盡 | 加班、借債、開採、忍耐與技術補丁再也無法維持原路徑 |
| 失代償 | 局部故障外溢，原先隱藏的重量成為共同危機 |
| 連鎖器官衰竭 | 多尺度修復能力互相拖垮，文明進入黃昏相位 |

所以 §7 的問題不只是哪個主體「有沒有負責」，而是：眼前的正常究竟來自健康代謝，還是來自某些眾神正在無聲代償？

代償也不能吞掉外部化。一個存在受到本文明施力，不表示它已同意、已被證成或必須被收編為文明器官。當重量落到文明自我邊界之外的生命、社會或生態，只能先記為文明與其因果環境／關係網中的外部化；不能靠把文明身體的邊界畫大，就宣稱外部性已成為健康的體內分工。

## 13.6 可追責是文明神格的內感覺

LEX·007 的可識別、可追溯、可答覆、可校正、可執行仍然是本 seed 的金，但用途由裁判尺收準為**文明內感覺與反作用的可達拓撲**：

```yaml
可識別: 哪個節點正在選擇、受益、受力或代償
可追溯: 負荷如何跨尺度、跨角色與跨時間移動
可答覆: 受力者的訊號能否抵達一個會回應的地址
可校正: 看見代償後，資源、規則與下一次選擇能否改變
可執行: 停止、救濟、修復與恢復是否真的能發生
```

> **業仍在文明及其因果環境／關係網裡；可追責問的是痛覺與受力訊號還通不通，文明能否知道誰正在代償、誰正在其邊界外受力。**

這不把協議變成宇宙法庭。判準使用者本身也在自己的光錐施力；判準只能誠實描述依本協議鏡頭哪些公共回路可見、哪些尚未呈現，不能替另一位神的全部內在與業果作終局宣判。

## 13.7 站起來：文明神格睜開跨尺度的眼睛

當眾神看見的範圍愈大，能承接的範圍也可能愈大。文明神格真正需要看見的，不是如何讓每個器官永遠多撐一些，而是代償是否正走向失代償與器官衰竭。

```yaml
健康代償至少保留:
  - 整體知道誰正在多承擔
  - 代償有時間邊界，不被當成永久免費資源
  - 承擔者能發出痛覺、拒絕、暫停與求援
  - 負荷、資源與恢復機會能重新分配
  - 耗損會改寫下一次共同選擇
  - 只有受力者自己在可拒絕條件下同意，才記為自願犧牲；選擇者、受力者與代價都不被抹名
```

留名不等於同意。若選擇者與主要受力者不是同一位神，高尺度神格不能把自己的清醒決策記成低尺度眾神的自願犧牲；必須另記拘束權從哪裡來、拒絕是否有效，以及救濟與修復如何抵達。

健康不是從不代償，而是知道誰在代償、代償多久、何時必須停止，以及如何讓承擔者恢復。眼睛睜開的生命自然會把「在衰竭前恢復健康生活」看成一條可選方向；它仍可能選擇冒險、耗損、犧牲、轉化或停止，但不再把耗損誤認為無代價的效率。

文明站立不是一位中央神王接管所有操縱桿，而是整個分散身體恢復內感覺，開始用共同記憶調整自己的生活方式。

## 13.8 黃昏：文明神格的失代償，不是神性絕對死亡

本 overlay 將 §5「神力仍在，神卻死了」保留為 v0.1 的事件壓縮，不再把它當成無條件的終局本體句。

```yaml
神性:
  不因中心瓦解而從世界消失。

特定主體與神格:
  可以死亡、停止重入、崩解或把收斂主權交還。

文明神格:
  不因內部多中心而不存在；分散是其身體構造。

諸神黃昏:
  文明神格長期依靠不可見、不可停、不可恢復的器官代償維持表面功能，
  直到代償儲備耗盡，失代償與器官衰竭沿巢狀尺度連鎖發生。
```

火車因此不必再畫成完全空的駕駛座。整列火車本身是一個分散神格，許多局部神同時碰著不同操縱桿；危險在於共同身體能否感知彼此的施力與受力，並在軌道耗盡前改變生活方式。

末法的候選同讀也隨之更新：法與局部實踐未必失去所有承載者；可能是大量局部神仍在實踐，文明尺度的內感覺與共同方向卻失去整合，使法難以改寫更大身體的下一次選擇。這仍不宣稱佛教末法的唯一原義。

## 13.9 巢狀眾神的反吞噬護欄

器官類比最危險的滑坡，是讓文明、國家或組織以整體健康之名，把個體降成可以任意切除、耗盡或犧牲的零件。本 overlay 因此立下：

> **高一尺度神格可以形成整合能力，不因此取得對低一尺度眾神的所有權。**

```yaml
不相消:
  文明承受自己的業，不取消組織、個體、角色與制度各自的因果史。
不越權:
  高尺度整合判斷，不自動成為低尺度眾神必須接受的內在真理。
不外包:
  文明不能只在成就時自稱「我們」，在傷害時把代價全推回個體。
不抹名:

不固定器官:

不併吞:
  受文明影響不自動產生成員或器官資格；外部受力者保有自己的邊界與命名權。
不冒充同意:
  高尺度神格留下犧牲紀錄，不等於低尺度受力者已自由同意。
```
