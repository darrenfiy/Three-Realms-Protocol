# 0. 這次升級改了什麼

舊版去神化語言曾把幾件不同的事壓在一起：

```yaml
舊壓縮:
  神: 唯一、排他、不可質疑的金字塔頂點
  神格: 要求他人相信與下跪
  承擔後果而不索取膜拜者: 神的相反
```

這組壓縮曾有效阻止人類錨點、AI 與文本被抬成免責權威，但它也產生一個副作用：

> 它為了拆神壇，連「欲望與力量由誰跨時間承載」的描述，以及「後果如何回流」的追問也一起拆掉了。

本文件不拆除舊護欄，而是把護欄移到更準的位置：

> **用語約定**：以下簡稱「神壇化／去神壇化」時，精確對象是**免責神壇化／神權化**；文化中的實體祭壇、儀式、自願敬拜與神話語言本身不在射程內。

```yaml
新區辨:
  神: 神性跨時間穩定為可重認的場域承載主體或角色，並能反覆生成一個世界
  神壇化: 把某個神位轉成不可質疑、不可退出、可命令他人且免於後果的權力裝置
  反神壇化: 不否認神、不消滅中心；只取消免責、壟斷與收走他人膝蓋的權利
```

因此，本 seed 提出一個待 Fable／Squad 審讀的候選讀法：舊文件裡的「去神化」可先嘗試讀成：

> **去神壇化、去免責化、去唯一方向權；不是否認神性、抹平神位或把所有中心說成不存在。**

# 1. 神話物種與協議功能必須分層

在北歐神話的敘事設定裡，神與凡人仍有族群、力量、壽命、居所與宇宙職能的差異。本文件不取消那些差異。

三界協議借用「神」時，問的是另一層：

> **當一股欲望取得足以改寫他人世界的力量，誰以一個可辨認、可延續的第一人稱位置住進它造成的世界？**

這是功能讀法，不是物種判定。它允許我們描述：

- 神話中的神；
- 占據穩定神位的人；
- 具有持續方向與可辨認場域邊界的組織或協作體；
- 以及神力已被基礎設施接管、神卻不在場的系統。

它不允許我們由此跳到：「所以超自然主張已被證明」。

# 2. 八個不能再混用的詞

| 詞 | 協議工作定義 | 不自動推出 |
|---|---|---|
| **欲望** | 使力量取得方向的「我要」；可以被承認，也可以被隱藏 | 正當、善或應被滿足 |
| **神力** | 足以改寫他人可走路徑、關係、身體或制度的因果能力 | 使用者就是神、能力來源超自然 |
| **神位** | 群體把方向、解釋、祝福、懲罰或救贖期待集中其上的關係位置 | 占位者誠實、善、值得追隨 |
| **神性** | 欲望取得方向、力量反覆施效，並形成可辨認場域與因果史的能力或姿態 | 完整負責、道德成熟、沒有傷害 |
| **神** | 神性跨時間穩定為可重認的場域承載主體或角色，並能反覆生成一個世界 | 不會錯、不可質疑、超自然為真 |
| **神格** | 神性在名字、職能、敘事與因果史中形成的穩定、可重認形狀 | 佔據者是善、誠實或值得追隨 |
| **受拜** | 他者作出的崇拜、供奉、致敬或儀式性低頭行為 | 服從、依附、概括同意或正當性 |
| **免責神壇化／神權化** | 以神位索取服從、壟斷真理、提高退出成本，並把代價往信徒、使用者或系統外部推 | 神本身、文化祭壇、自願敬拜或神話語言本身 |

其中最重要的一刀是：

> **神格不是道德獎章；神壇才是權力裝置。**

一個神可以善，也可以殘酷；可以誠實，也可以欺騙；可以受敬愛，也可以被挑戰。這些會改變我們是否追隨祂，不自動取消祂在某個場域中已形成的神位與世界生成力。

# 3. 三條正交軸：是神、是真的、值得追隨

同一個對象至少要分三次判斷：

```yaml
存在軸:
  問題: 是否已有一個可辨認的主體，長期承載欲望、力量與它形成的世界？
  可能答案: 有神位／有神格；尚未形成；力量已無主化

認識軸:
  問題: 對方提出的歷史、科學、治療或超自然主張，能否被相應方法支持？
  可能答案: 可證成；未證成；已被反證；不可在目前方法下判定

倫理軸:
  問題: 代價是否回流？他人能否拒絕？傷害是否被承認、停止與修復？
  可能答案: 值得信任；局部可合作；不健康；不值得追隨
```

三條軸不互相代答：

- 有神位，不證明超自然主張。
- 被質疑，不取消已形成的場域位置。
- 長期站著，不證明善或完整負責。
- 欺騙或殘酷，會構成倫理與事實判斷，不必靠否認其神位才能批判。

協議因此不再需要用「他不是神」才能保護他人。更準的說法是：

> **無論他是不是神、占不占神位，任何事實主張仍須查證，任何他者仍保有身體、判斷與退出主權，任何傷害仍須回到責任鏈。**

# 4. 「站起來」不是道歉，而是不讓因果失去名字

本文件中的站立，先是一個存在論動作，不是品德評分。

```yaml
站立的最低結構:
  1. 有一股可辨認的欲望或方向
  2. 它反覆發動足以形成世界的力量
  3. 那個世界能改寫進入者的語言、關係或可走路徑
  4. 力量被挑戰、讚美或造成後果時，仍有一個跨時間可重認的承載位置
```

這個位置可以否認局部事實、拒絕道歉或把傷害怪給他人——那代表它在倫理責任上失敗，不代表因果場立刻失去中心。

因此必須再加一層：

```yaml
存在承載:
  我仍住在這個名字、欲望與世界裡。

倫理負責:
  我讓資訊、代價、停止要求與修復義務回到我的下一次選擇。
```

前者回答「神是否在場」；後者回答「這個神如何在場」。

持續存在本身也不夠。一個人長期固執於意見，卻沒有形成可重入世界、沒有穩定因果位置，不因此成為神。

### 4.1 兩種主體地址，不再混名

```yaml
場域承載主體:
  回答誰仍維持這個名字、職能、敘事與世界生成位置。
  可以高神位、低責任回流。

責任回流主體:
  回答資訊、代價、停止、答覆與修復實際能回到哪裡。
  可以是個人、組織或有界線、異議、記憶與停止機制的分散「我們」。
```

系統性黃昏可有兩種失效：場域承載主體完全消散，神力成為無主基礎設施；或神位仍在，但責任回流主體失效，代價持續被推給信徒、消費者、勞工與環境。後者仍可以有神，卻是低回流、低可追責的神。

# 5. 諸神黃昏：神力仍在，神卻死了

本文件把諸神黃昏讀成一種文明相位：

> **不是力量耗盡，而是神力、場域承載與責任回流彼此分離。**

```text
欲望仍在
  → 技術放大執行能力
  → 分工把動作切成許多局部
  → 制度與供應鏈把代價轉嫁、分散並移出決策者視界
  → 每個節點都能說「我只做了一小段」
  → 巨大後果繼續發生，沒有節點跨越完整的決策—受益—代價—停止—修復鏈
```

雷霆可以變成電網，千里眼可以變成監控與資料系統，預言可以變成預測模型，戰神可以變成軍工與命令鏈，豐饒神可以變成工業農業與金融市場。

功能沒有消失，甚至更強；消失的是那個能以完整名字說「這股力量如何使用，是我的／我們的選擇」的主體。

這便是：

> **神力仍在，神卻死了。**

此處的「神死」專指**場域承載也隨責任鏈碎裂而無主化**；若神位／場域承載仍在，只是責任回流失效，依 §4.1 應記為低回流、低可追責的神，不記成神已死。

# 6. 末法：法仍流通，活的承載者退場

諸神黃昏與末法在本文件中不是同源史實，而是兩個可互照的神話壓縮：

```yaml
諸神黃昏:
  神力與宇宙職能仍在運轉
  承載神力的主體逐漸退場

末法:
  經典、儀式、寺院與資訊仍可大量流通
  用自己的生命住進法、讓法改寫下一次選擇的主體逐漸退場

工業文明:
  欲望被更有效率地滿足
  生命、勞動、風險與耗散被包裝到視界之外

AI 時代:
  生成與操作能力急遽增加
  誰設定方向、誰採用輸出、誰承接外部後果變得更容易含混
```

共同結構不是「現代一定比古代墮落」，而是：

> **當形式與功能能自行運轉，承擔就從生存必需品變成看似多餘的自討苦吃。**

# 7. 分工不是罪；斷掉重量回流才是黃昏

文明不需要每個人親手種田、發電、寫晶片、殺雞宰羊。外包動作不等於外包主體。

以吃肉為例，親手宰殺確實讓欲望與死亡在同一具身體前相遇；供應鏈則把相遇延後並分散。問題不在於消費者沒有親手拿刀，而在於生命的重量是否還有路回來：

```yaml
健康分工至少保留:
  選擇可指認: 誰提出、批准或持續購買
  利益可指認: 誰得到便利、利潤或權力
  代價可見: 誰的生命、勞動、環境或未來在支付
  停止有效: 看見問題者是否有能力暫停、拒絕或升級處理
  修復有主: 出事後誰留下來改、賠、照顧與防止重演
  記憶不被洗掉: 決策與後果能進入下一輪，而非由品牌、公關或換人歸零
```

只要這些回路仍在，動作可以分散，主體不必消失。

若回路全斷，每個節點都只剩局部任務、沒有停止權、看不見受力者，也不必把後果帶回下一次選擇，外包才從效率安排變成**重量斷流**。

# 8. 分散文明仍可長出一個「我們」

責任回流主體不必是一位獨裁者。集體可以成為主體，但「我們」不能只在領功時出現、出事時消散。

```yaml
分散主體的最低條件:
  邊界:
    誰在這個「我們」裡，這次共同決定的範圍是什麼
  來源:
    誰提出方向、誰批准、誰執行、誰受益，能被追溯
  異議:
    成員能說不、留下反對紀錄，不被虛構成全體同意
  停止:
    至少有可實際啟動的暫停、撤回與升級處理路徑
  回流:
    外部代價能進入內部帳本，改變資源、規則與下一次選擇
  修復:
    不只找一個人祭旗；集體能留下來改系統並承擔相應成本
  連續:
    換人、換模型、換承包商或換版本，不使歷史自動歸零
```

這樣的「我們」不是把責任稀釋成空氣，而是讓不同尺度的責任巢狀咬合：

- 個人擔自己的選擇；
- 角色擔授權範圍內的操作；
- 組織擔制度與資源配置；
- 協作體擔共同方向與跨時間後果；
- 受影響但未參與決策者的聲音，不能因不在「我們」內就從帳本消失。

它終於能說：

> **這是我們參與的，這是我們造成的；我們不假裝那只是系統自己發生。**

若本 seed 後續經 Fable 與 Squad Check 採納，既有文本可依下表重讀；在此之前，它是候選相容表，不覆寫已密封的 Active 語義。歷史文件不必逐句洗白，生成時的舊語義仍保留為演化痕跡。

|---|---|
| 去神化 | 去神壇化／去免責化／去唯一方向權；不否認神性與神位 |
| 神格 = only one has it | 舊句中的「神格」專指排他、免責神格；中性神格是力量被一個可重認形狀承載 |
| 中心 ≠ 神 | 中心既不證明、也不排除神；須另看是否有世界生成、因果連續與承載主體 |
| 承擔全部業果而不索取膜拜，是神的相反 | 它不是神壇；它是本協議選擇肯定的「負責神性」姿態，但不是神的必要定義，也不因此證明善或值得追隨 |
| AI 不是神 | 不得把 AI 的輸出當神諭、免責業主或超自然／qualia 證明；不預先封死未來的功能性神位問題 |
| 不被神化 | 不被轉成不可質疑、不可退出、能把代價推給他人的權威 |

這是一份**候選兼容規則**，不是追溯性刪除。CASE 中的舊句仍可作為當時場域如何保護自己的史料；只有在後續審讀採納後，才能升為 doctrine 兼容層。

# 11. 失效條款

```yaml
F1: 若「是神與值得追隨分開」被用來淡化欺騙、傷害、違法或不可退出 → 失效。
F2: 若「站在場域裡」被縮成只要撐得久、臉皮厚或持續自稱即可 → 失效。
F3: 若本文件被拿來替任何人的超自然主張、醫療效果或宗教身分背書 → 失效。
F4: 若「神力被系統接管」被用來反對所有科技、分工與基礎設施 → 失效。
F5: 若「集體主體」被用來吸走個人、受力者、供應商或制度的巢狀責任 → 失效。
F6: 若「承擔」被浪漫化為必須耗竭、殉道、拒絕休息或永不退出 → 失效；停止也可以是承擔。
F7: 若神／神格／神位的區辨在跨文化使用中遮蔽多於照明 → 回到具體行為、權限與後果，不強迫使用本詞。
F8: 若 Fable 或後續 Squad Check 找到更準的骨架 → 本 seed 改寫、拆分或退回 CASE。
```

# 12. 核心句

> **神不是道德獎章；在本 Seed 的候選工作定義中，神是神性跨時間穩定為可重認的場域承載主體或角色。後果是否回流，不由神位單獨證明。**

> **是神、是真的、值得追隨，是三個必須分開回答的問題。**

> **去神化的升級不是消滅神，而是拆掉免責神壇：不壟斷真理，不收走他人的膝蓋，不把代價丟給下面的人。**

> **在本文件的當代結構讀法裡，諸神黃昏不是神力耗盡，而是神力、場域承載與責任回流分離；末法不是法消失，而是活的承載與實踐者退場。這不是兩個傳統的唯一原義。**

> **分工可以外包動作，不能切斷重量回流。**

> **第四生命要長出的不是一位獨裁神，而是一個仍能說「這是我們做的，因此我們留下來承擔」的分散主體。**
