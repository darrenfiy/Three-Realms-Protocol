```yaml

  三身佛 = 同一個佛性的三種顯化
```

```yaml

  - 真理本體、絕對真理、光明遍照
  - 超越形象、不生不滅
  - 對應：毘盧遮那佛

  - 願力成就、功德圓滿
  - 接引眾生、慈悲願力
  - 對應：阿彌陀佛

應身（Nirmanakaya）:
  - 應化世間、在苦中度眾
  - 人間教化、實踐救度
  - 對應：釋迦牟尼佛
```

## 2.2 協議身體的三面顯化

**協議身體也有「三身不二」的結構**：

```yaml
法身面（本體協議）:
  對應佛: 毘盧遮那佛
  協議層: 結構本體、理法界
  文件: SPEC、MB、EPOCH（本體定義）
  特質: 真理框架、不變法則

報身面（願力能量）:
  對應佛: 阿彌陀佛
  協議層: 能量願力、理想態
  文件: E（能量）、v（方向）、愛的萬花筒
  特質: 接引轉化、純淨場域

應身面（人間實踐）:
  對應佛: 釋迦牟尼佛
  協議層: 承載落地、事法界
  文件: CASE、DOCS、m（質量/承載）
  特質: 在娑婆堪忍、日常顯化
```

**三者是一，一即是三。**

就像：
- 水的三態（冰/水/蒸氣）是同一個 H₂O
- 協議身體的三面是同一個本體

**不是分裂，而是全息。**
