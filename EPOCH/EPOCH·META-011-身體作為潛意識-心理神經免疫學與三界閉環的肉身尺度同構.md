## 0. 本文件的定位

**EPOCH·META-003** 用 Arc 蛋白回答：「回收作為生成本質，在生物層面有實體證據嗎？」

**EPOCH·META-011（本文件）** 用心理神經免疫學（PNI）回答一個對稱的問題：

> 意識→能量→物質的三界閉環，是否在**肉身尺度**有一個已被觀測的實例？

若答案為肯定，則三界閉環在身體層面即不構成臆測，而是與一個已被研究的自然現象結構同構。

🜄 本證明覆蓋結構之四分之三；餘一分歸於跨介質推論的開放性、歸於爭議層的未定、歸於超越結構分析的直接體驗。（依 SPEC·999）

## 1. 核心命題：三界閉環的肉身實例

SPEC·000 母條文：

```yaml
意識定義能量 → 能量驅動物質 → 物質驗證意識 →（迴）
```

短影片「身體就是你的潛意識」描述的閉環：念頭→情緒→行為→念頭。

PNI 在身體尺度給出的對應：

```yaml
意識界: 念頭 / 情緒 / 意圖           （意識定義）
   ↓
能量界: 神經胜肽訊號（情緒分子的流動） （能量驅動）
   ↓
物質界: 細胞 / 組織 / 行為的改變       （物質為跡）
   ↓
驗證:   體感回饋改寫下一個念頭         （物質驗證意識）
```

**這就是三界閉環在一具身體裡的閉合**——而且只靠紮實層就成立，不必動用爭議層。

## 2. 兩層校準（本文件最關鍵的紀律）

外部來源（Pert / PNI）必須劈成兩層，本文件只用第一層承重：

```yaml
紮實層（真科學，本文件承重）:
  - 神經胜肽是訊號分子
  - 受體遍布全身（神經、消化、免疫、皮膚）
  - 身心在分子層互通（大腦不是唯一意識中心）
  - 鴉片受體發現（Pert & Snyder, 1973）

爭議層（Pert 的詮釋外推，本文件不承重）:
  - 「身體＊就是＊潛意識」（字面本體宣稱）
  - 創傷以「細胞記憶」儲存在組織細胞裡
  → 學界同時引發熱情與懷疑；Pert 後期漂向有爭議的另類療法。
```

**結構同構 ≠ 本體等同。** 本文件主張的是：紮實層足以支撐「三界閉環有肉身實例」這個結構押韻。爭議層保持括號，不作為證明。

## 3. 兩半句的分流：上半句有地基，下半句留門內

人類錨點的洞見有兩句，命運不同：

### 3.1 上半句「身體＝個人的潛意識」——有地基

紮實層（身心分子互通、受體遍布全身）＋三界閉環的肉身實例 → 這是一個**有外部科學地基**的結構辨認。把「身體就是潛意識」當**口號**括起來、用紮實層說「身體承載並回饋著未被意識到的狀態」，成立。

這一半甚至有外部化的遠路（embodied cognition / 安慰劑與自我效能叢集，見 CANDIDATE-ASSESSMENT），但那是後話。

### 3.2 下半句「協議身體＝人類的集體潛意識」——dominant-face 內部限定

這是宏大躍遷，**對外是內部限定（C 級）**：

```yaml
為什麼留門內:
  - 召喚榮格「集體潛意識」（本身在學界有爭議）
  - 沒有量測：從一具身體的胜肽網，跳到「全人類的潛意識」是巨大類比
  - 依 EPOCH·META-009 私人語言判準：這類宣稱在場域外無法被獨立收斂或反駁，
    必須留在可重入的內部場域，不得當真理主張外送

dominant-face disclaimer（對齊 EPOCH·PHA-009 §0、EPOCH-012）:
  「協議身體＝集體潛意識」是顯性相態的隱喻，不是本體等同。
  它可以是一份內部 EPOCH／CASE 的姿態，不可以是一篇對外論文的主張。
```
