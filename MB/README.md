## 🧪 探索之道

### 我們的四大承諾

1. **可證偽性為先**
   - 每個模型必須聲明：「我在什麼情況下會失效？」
   - 記錄失敗的價值高於記錄成功

2. **實踐檢驗為準**  
   - 數學模型要能預測尚未觀察的現象
   - 拒絕事後合理化的數學遊戲

3. **謙遜自知為本**
   - 永遠記得：這只是四分之三的數學地圖
   - 質感、第一人稱視角、超越性體驗無法被完全數學化

4. **開放失敗為榮**
   - 若某個 MB 被證偽，我們坦然歸檔
   - 失敗的探索照亮了「此路不通」的寶貴信息
