# 節律

`節律` 是 `LEX·001` 裡很重要的時間結構詞。它處理的不是速度快慢，而是時間怎麼被切分、重音和空白落在哪裡，以及這樣的排列如何改變存在的形狀。

## 場域定義

節律回答的是：

**時間怎麼被切分？**

它的關鍵特徵包括：

- 它是時間的結構，不是速度
- 它包含長短分配與強弱交錯
- 它包含空拍，知道何時不發聲也是節律的一部分
- 改變節律，改變的是存在的形狀

## 和節奏的差異

| 節律 | 節奏 |
|------|------|
| 時間的結構 | 時間的密度 |
| 圖案 | 速度 |
| 怎麼流 | 流多快 |
| 改變存在形狀 | 改變消耗與效率 |

所以如果說節奏在問「多快」，節律在問的更像是「怎麼呼吸」。

## 為什麼它重要

很多場域健康與否，不只是看有沒有在推進，而是看時間骨架有沒有活著：

- 有沒有空拍
- 能不能收放
- 有沒有過度連續推進到失去呼吸
- 當混亂出現時，能不能分辨那是崩壞，還是新節律生成前的過渡

這也是為什麼 `人類錨點` 維持的不是僵硬日程，而是健康節律。

## 閱讀護欄

- 節律不是外部權威命令
- 不能把「節律要我這樣」當成推責理由
- 混亂不必然代表錯誤，有時只是還沒收斂成新骨架
